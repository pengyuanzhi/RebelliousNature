# 第一百零一章 玄天宗的日常

时间:加入玄天宗三个月后
地点:玄天宗
人物:秦寒、苏清歌、赵灵儿

加入玄天宗已经三个月了。

这三个月来,秦寒适应了中界的生活节奏。

每天早晨修炼两个时辰,然后接取宗门任务,下午继续修炼,晚上和苏清歌双修。

这样的日子虽然单调,但修为提升得很快。

"金丹后期后期...距离元婴期只有一步之遥了。"秦寒感叹道。

他盘坐在洞府的蒲团上,感受着体内的金丹。

金丹已经完全凝实,散发着璀璨的金光。只要再进一步,就能凝结元婴。

"但这一步,却是最难的。"秦寒暗道。

元婴期是修真路上的一个重要分水岭。

金丹期修士虽然强大,但在修真界只能算是中层。而元婴期修士,就是真正的高手了。

"必须尽快突破到元婴期。"秦寒暗道,"只有这样,才有能力去魔界救父亲。"

就在这时,洞府外传来了敲门声。

"请进。"秦寒说道。

门开了,苏清歌走了进来。

"秦寒,今天要完成宗门任务吗?"她问道。

"对。"秦寒点头,"你这个月的任务完成了吗?"

"完成了。"苏清歌笑道,"我帮你把草药都采齐了。"

"辛苦你了。"秦寒说道,"我也完成了任务,猎杀了一头五阶妖兽。"

"秦寒真厉害。"苏清歌赞叹道,"才金丹后期,就能猎杀五阶妖兽。"

"五阶妖兽相当于元婴初期。"秦寒解释道,"不过我用了命轮眼,才能险胜。"

两人聊了一会儿,赵灵儿也来了。

"师兄,师娘。"灵儿乖巧地叫道。

这三个月来,灵儿的适应得很好。

她不仅学会了说话,还学会了识字读书。而且她的修为也在稳步提升,已经达到五阶中期,相当于元婴中期!

"灵儿,你的修炼如何?"秦寒问道。

"很好。"灵儿笑道,"玄天宗的功法很强,我学到了很多。"

"而且,我还认识了不少朋友。"

"那就好。"秦寒欣慰地点头。

灵儿虽然化形不久,但天赋极高,而且性格活泼可爱,在玄天宗很受欢迎。

"对了。"灵儿说道,"听说宗门要举行洞天大比?"

"洞天大比?"秦寒疑惑道。

"对。"苏清歌解释道,"玄天洞天每十年举行一次大比,所有宗门都会参加。"

"大比的前十名,可以进入洞天秘境,获得大量机缘。"

"原来如此。"秦寒点头,"那我们都要参加。"

"嗯。"苏清歌和灵儿都点头。

玄天宗的日子,平静而充实。

秦寒知道,这样的日子不会持续太久。

因为父亲还在魔界等着他去救。

# 第一百零二章 宗门任务

时间:当日
地点:玄天宗任务堂
人物:秦寒、苏清歌、赵灵儿

三人来到任务堂,准备接取新的宗门任务。

任务堂是玄天宗发布任务的地方。

弟子们可以在这里接取任务,完成后可以获得灵石、功法、法宝等奖励。

"秦师兄,苏师姐,赵师姐。"任务堂的执事笑着打招呼,"今天要接什么任务?"

"看看有什么高级任务。"秦寒说道。

执事拿出一个玉简,上面列着所有高级任务:

1. 猎杀六阶妖兽:奖励灵石十万,元婴丹一颗。
2. 寻找混沌原石:奖励灵石五万,神阶法宝一件。
3. 探索上古遗迹:奖励灵石二十万,圣阶功法一部。
4. 护送商队:奖励灵石三万,聚灵丹十颗。
5. 炼制七品丹药:奖励灵石十万,炼丹经验。

"六阶妖兽..."秦寒思索道,"六阶妖兽相当于元婴后期,太危险了。"

"混沌原石?"苏清歌说道,"这个倒是我们需要的。"

"不过,混沌原石极其罕见,不知道哪里能找到。"

"上古遗迹..."灵儿眼睛一亮,"这个听起来很有意思。"

秦寒想了想,说道:"那就接探索上古遗迹的任务吧。"

"正好可以试试我的实力,也能获得一些机缘。"

"好。"执事点头,"上古遗迹在玄天洞天的东部,据说是一座上古洞府。"

"不过要注意,遗迹中有很多机关和妖兽,要小心。"

"我会的。"秦寒说道。

接取任务后,三人准备出发。

就在这时,一个声音传来:"等等,我也要去。"

秦寒回头,看到一个身穿紫袍的女子。

女子约莫二十岁,容貌绝美,气质高冷。

"你是?"秦寒问道。

"慕容雪。"女子淡淡地说道,"玄天宗圣女。"

"慕容雪..."秦寒重复了一遍这个名字,"久仰大名。"

慕容雪是玄天宗的天才,据说十八岁就达到了元婴初期,是玄天宗年轻一辈的最强者。

"你也去上古遗迹?"秦寒问道。

"对。"慕容雪点头,"我对上古遗迹很感兴趣。"

"那一起吧。"秦寒说道,"多一个人,多一份力量。"

慕容雪看了秦寒一眼,淡淡地说:"希望你不要拖后腿。"

说完,她转身就走了。

"这性格..."苏清歌有些尴尬,"好像不太好相处。"

"没关系。"秦寒笑道,"只要实力强就行。"

三人离开任务堂,准备前往上古遗迹。

而慕容雪,也跟在后面。

# 第一百零三章 结识慕容雪

时间:出发路上
地点:玄天洞天东部
人物:秦寒、苏清歌、赵灵儿、慕容雪

飞行在云端,秦寒打量着慕容雪。

她是玄天宗的圣女,天赋极高,十八岁就达到了元婴初期。

现在应该已经元婴中期了。

"慕容仙子,你为什么也要去上古遗迹?"秦寒问道。

"关你什么事。"慕容雪冷冷地说道。

"我只是好奇。"秦寒苦笑。

慕容雪沉默了片刻,然后说道:"我在寻找一部功法。"

"什么功法?"

"太上忘情录。"慕容雪说道,"这是我玄天宗失传的圣阶功法。"

"据传,这部功法在上古遗迹中。"

"太上忘情录..."秦寒重复了一遍,"听起来很强。"

"当然。"慕容雪傲然道,"圣阶功法,岂是弱者能修炼的。"

秦寒没有反驳,只是笑了笑。

一行人继续飞行,很快就到了上古遗迹的入口。

遗迹是一座古老的洞府,入口处被阵法封印着。

"这阵法..."慕容雪皱眉,"很难破解。"

"让我试试。"秦寒上前。

他开启命轮眼,观察阵法的结构。

在命轮眼的视野中,阵法的构造清晰可见。

"找到了。"秦寒说道,"阵眼在那里。"

他一指点出,击中阵眼。

阵法的光芒闪烁了几下,然后消失了。

"竟然破解了..."慕容雪有些惊讶,"你是什么境界?"

"金丹后期。"秦寒老实说道。

"金丹后期?"慕容雪不敢相信,"怎么可能..."

"我有特殊的手段。"秦寒没有解释太多。

"走吧,我们进去。"

四人进入遗迹,发现里面是一座巨大的宫殿。

宫殿中有很多房间,每个房间都有不同的宝物。

"我们分头找吧。"慕容雪说道,"我找太上忘情录,你们找其他的。"

"好。"秦寒点头。

四人开始探索遗迹。

秦寒和苏清歌、灵儿一组,慕容雪一组。

很快,秦寒就发现了一些有用的东西:灵草、矿石、丹药...

"这些收获真不错。"苏清歌欣喜道。

"嗯。"秦寒点头,"继续找,说不定还有更好的。"

就在这时,遗迹深处传来一阵巨大的声响。

"轰隆——!"

"不好!"秦寒脸色一变,"有人触发了机关!"

# 第一百零四章 从对手到朋友

时间:遗迹探险中
地点:上古遗迹深处
人物:秦寒、苏清歌、赵灵儿、慕容雪

秦寒三人朝着声响的方向跑去。

很快,他们来到了遗迹的最深处。

那里,慕容雪正被一只巨大的傀儡围攻。

傀儡有六阶实力,相当于元婴后期。

而慕容雪虽然元婴中期,但在傀儡的围攻下,渐渐落入下风。

"该死..."慕容雪咬牙坚持,但伤口越来越多。

"我来帮你!"秦寒冲了上去。

苦海命运剑发出璀璨的金光,一剑斩在傀儡身上。

"嗤——!"

傀儡被斩出一道深痕,但很快又恢复了。

"这傀儡会自我修复!"苏清歌惊呼。

"那就攻击它的核心!"秦寒说道。

他开启命轮眼,寻找傀儡的弱点。

很快,他就找到了:傀儡的核心在胸口处,有一颗灵石。

"找到了!"秦寒一剑刺向傀儡的胸口。

"啪!"灵石碎裂。

傀儡的身体瞬间崩溃,化作一堆废铁。

"多谢。"慕容雪喘着气说道,"如果不是你,我恐怕就危险了。"

"不客气。"秦寒笑道,"大家都是同门,应该互相帮助。"

慕容雪看着秦寒,眼中闪过一丝异色。

她一直以为自己是最强的,没想到秦寒一个金丹后期,竟然能击杀六阶傀儡。

"你的实力,很强。"慕容雪说道。

"还行。"秦寒谦虚地说道,"只是运气好罢了。"

"不只是运气。"慕容雪摇头,"你有一种特殊的能力,能看穿弱点。"

"这是命轮眼。"秦寒没有隐瞒,"我的特殊能力。"

"命轮眼..."慕容雪喃喃自语,"传说中能看破命运的眼睛..."

"没错。"秦寒点头。

慕容雪沉默了片刻,然后说道:"我欠你一个人情。"

"以后如果有需要,可以找我。"

"好。"秦寒笑道。

慕容雪的性格虽然高冷,但还是有恩必报的。

"对了,你找到太上忘情录了吗?"秦寒问道。

"没有。"慕容雪摇头,"这部功法可能不在这里。"

"不过,我找到了另一部功法:《步天诀》。"

"《步天诀》?"苏清歌惊喜道,"这不是传说中的身法吗?"

"对。"慕容雪点头,"圣阶身法,威力强大。"

"不过,我不需要身法,你们谁要?"

"我要。"秦寒说道,"正好我缺一部好的身法。"

"给你吧。"慕容雪拿出玉简,递给秦寒。

"多谢。"秦寒接过玉简。

"不客气。"慕容雪说道,"就当是报答你的救命之恩。"

四人离开遗迹,返回玄天宗。

一路上,慕容雪的话多了起来。

虽然还是高冷,但对秦寒的态度明显好了很多。

秦寒也能感受到,慕容雪是一个外冷内热的人。

只要成为她的朋友,她就会真心对待。

"也许,我们能成为朋友。"秦寒暗道。

# 第一百零五章 洞天大比

时间:返回玄天宗后
地点:玄天宗
人物:秦寒、苏清歌、赵灵儿、慕容雪

回到玄天宗后,宗主召集了所有核心弟子。

"三个月后,洞天大比就要开始了。"宗主说道。

"这次大比,所有宗门都会参加。我们玄天宗,要派出最强的弟子。"

"所以,我要从你们中选出十人,代表玄天宗出战。"

"秦寒、慕容雪、苏清歌、赵灵儿..."

宗主一一念出名字。

"你们十人,要好好准备。三个月后,为玄天宗争光!"

"是!"众人齐声应道。

散会后,慕容雪找到秦寒。

"秦寒,你有信心吗?"她问道。

"还行。"秦寒笑道,"不过,我想知道大比的规则。"

"规则很简单。"慕容雪解释道,"所有参加大比的弟子,进行单挑。"

"最后决出前十名,前十名可以进入洞天秘境。"

"秘境中有很多机缘,对我们的修炼很有帮助。"

"原来如此。"秦寒点头,"那看来要全力以赴了。"

"嗯。"慕容雪说道,"我听说其他宗门也有很多天才。"

"比如剑宗的剑狂、丹塔的丹仙、药王谷的神医..."

"都是很强的对手。"

"不怕。"秦寒笑道,"强者越多,越有挑战性。"

慕容雪看了秦寒一眼,眼中闪过一丝赞赏。

她发现,秦寒和一般的修士不同。

他不畏惧强者,反而渴望挑战。

这种性格,她很欣赏。

"三个月后,看看谁更厉害。"慕容雪说道。

"好。"秦寒点头。

两人相视而笑,都燃起了竞争的火花。

接下来的三个月,秦寒开始全力修炼。

他的目标是:在大比中进入前十,然后进入秘境,获得更多机缘。

同时,也要争取突破到元婴期。

只有这样,才有足够的力量去魔界救父亲。

# 第一百零六章 大比开始

时间:三个月后
地点:玄天洞天中央广场
人物:秦寒、苏清歌、赵灵儿、慕容雪、各宗门天才

三个月后,洞天大比开始了。

玄天洞天的中央广场,聚集了来自各宗门的弟子。

总共有三十六个宗门,每个宗门派出十人,总共三百六十人。

"大家好!"一个身穿金袍的老者大声说道。

"我是玄天洞天的洞主,这次大比的主持者。"

"规则很简单:单挑,败者淘汰,胜者晋级。"

"最后决出前十名,前十名可以进入洞天秘境。"

"现在,大比开始!"

第一轮比赛开始了。

秦寒的对手是一个筑基后期的修士。

"请指教。"修士行礼道。

"请指教。"秦寒也行礼。

比赛开始!

修士率先出手,使出一套剑法,剑气纵横。

但秦寒只是轻轻一挥手,剑气就全部消散了。

"你..."修士大惊。

"该我了。"秦寒一指点出,修士就被击飞出场外。

"秦寒胜!"裁判宣布。

接下来的比赛,秦寒一路顺风顺水。

无论是筑基期,还是金丹期,在他面前都撑不过几招。

很快,他就进入了前十。

而其他人的表现也很精彩。

慕容雪依然强势,每一场都轻松获胜。

苏清歌和赵灵儿也进入了前十。

还有其他宗门的天才:剑宗的剑狂、丹塔的丹仙、药王谷的神医...

都是很强的对手。

"前十名诞生了!"洞主宣布,"接下来,进行决赛!"

"决赛规则:车轮战,最后站着的人,就是第一名!"

"开始!"

决赛开始了。

第一场:剑狂vs丹仙。

剑狂使出剑法,剑气纵横。丹仙则用丹药攻击,毒烟弥漫。

最终,剑狂险胜。

第二场:神医vs秦寒。

神医用毒术攻击,毒雾缭绕。

但秦寒有命轮眼,能看穿毒雾的弱点。

他一指点出,击碎神医的毒术,然后击败了他。

接下来的比赛,秦寒一路过关斩将。

击败了剑狂、丹仙、慕容雪...

最终,他站在了决赛的最后。

"最后一场:秦寒vs慕容雪!"洞主宣布。

两人的目光在空中交汇,都燃起了战意。

"来吧。"秦寒笑道。

"好。"慕容雪点头。
