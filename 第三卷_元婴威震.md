# 第一百五十一章 元婴初期的力量

时间：出关后第一天
地点：玄天宗·苦海峰
人物：秦寒

清晨，阳光洒在苦海峰上，灵气浓郁。

秦寒站在观景台上，感受着元婴初期的力量。

"元婴初期，果然强大。"秦寒握紧拳头。

他神识外放，覆盖方圆百里。每一处风吹草动，都尽收眼底。

"神识范围是金丹期的十倍。"秦寒满意地点头。

他催动元婴，元婴从丹田中飞出，在体内流转。

"元婴是修士的第二次生命。"秦寒喃喃自语，"即使肉身被毁，只要元婴逃脱，就能夺舍重生。"

他试着让元婴出窍。

元婴化作一道流光，从眉心飞出，悬浮在空中。

元婴与秦寒的本体一模一样，只是只有三寸大小，散发着淡淡的金光。

"元神出窍！"秦寒控制着元婴，在洞府中穿梭。

元婴的速度极快，转眼就遍历了整个洞府。

"有了元神出窍的能力，可以探查敌情，也可以偷袭。"秦寒发现元神有很多用途。

他收回元神，继续感受元婴初期的力量。

灵力比金丹期精纯数倍，力量也大幅提升。

"现在的我，可以轻松击败元婴初期的修士。"秦寒自信道。

"甚至，与元婴中期也能一战。"

就在这时，洞天真人来到苦海峰。

"出关了？"洞天真人问道。

"是的，师父。"秦寒行礼。

"让我看看你的修为。"洞天真人神识探入秦寒体内，片刻后，他脸上露出赞赏，"元婴初期，根基扎实，不错。"

"都是师父教导有方。"

"不必谦虚。"洞天真人摆手，"三个月突破元婴，而且根基如此扎实，你是我见过最有天赋的弟子。"

"师父过奖了。"

"对了，"洞天真人说道，"你现在元婴初期，可以接一些更高级的任务了。任务堂有一些元婴期的任务，你可以去看看。"

"是，师父。"

洞天真人离开后，秦寒前往任务堂。

# 第一百五十二章 洞天新任务

时间：出关后一周
地点：玄天宗任务堂
人物：秦寒、任务堂执事

秦寒来到任务堂，走到柜台前。

"这位长老，请问有什么可以帮您？"执事恭敬地问道。

出示长老令牌后，执事的态度更加恭敬。

"原来是秦长老。"执事说道，"秦长老现在是元婴初期，可以接取元婴期的任务了。"

"有哪些元婴期的任务？"秦寒问道。

执事拿出一个玉简，递给秦寒。

"这是元婴期任务列表，长老可以看看。"

秦寒神识探入玉简，查看任务列表。

"猎杀元婴初期魔兽，一只奖励一万上品灵石。"
"探索二级秘境，奖励五万上品灵石。"
"探索三级秘境，奖励十万上品灵石。"
"护送商队到邻界，奖励三万上品灵石。"

秦寒心中盘算。

"探索三级秘境，奖励十万上品灵石。"秦寒眼中闪过一丝兴趣。

不过，三级秘境危险性很高，可能有元婴后期的魔兽，甚至化神期的妖兽。

"我现在的实力，探索三级秘境有些勉强。"秦寒暗道。

"还是先探索二级秘境吧。"

"我接这个任务：探索二级秘境'暗影秘境'。"秦寒说道。

"好的，长老。"执事登记后，递给秦寒一个令牌，"这是任务令牌，探索秘境后，收集一些珍贵材料，就可以提交任务。"

"多谢。"

秦寒接过令牌，转身离开。

"暗影秘境..."秦寒查看任务详情。

暗影秘境是一处二级秘境，内有各种元婴期的魔兽和妖兽，还有很多珍贵的灵草和材料。秘境内部光线昏暗，能见度很低，非常危险。

"不过，也是很好的历练机会。"

秦寒返回洞府，准备出发。

他收拾好行装，带足了丹药和符箓，然后骑着苦海飞舟，前往暗影秘境。

# 第一百五十三章 秘境探险

时间：接任务后
地点：暗影秘境
人物：秦寒、秘境魔兽

暗影秘境位于玄天洞天东部，是一片阴暗的森林。

秦寒降下飞舟，小心翼翼地进入秘境。

刚进入，他就感受到一股阴冷的气息。

"这里果然阴暗。"秦寒开启命轮眼，神识全开。

命轮眼在黑暗中也能看清，秦寒观察到周围的一切。

秘境内，树木高大，遮天蔽日。森林中弥漫着浓雾，能见度极低。

"难怪叫暗影秘境。"秦寒暗道。

他小心翼翼地前进，神识不断搜索周围。

就在这时，他感受到一股气息从左侧传来。

"有魔兽！"

秦寒转头望去，只见一只黑色的豹子从树丛中窜出。

"暗影豹，元婴初期！"秦寒认出这只魔兽。

暗影豹速度极快，转眼就到了秦寒面前。

"来得好！"

秦寒不慌不忙，催动黑铁灵甲，防御提升。

暗影豹的利爪抓在灵甲上，发出刺耳的声音。

"好强的力量！"

秦寒催动《苦海经》，全身力量爆发。

"剑气斩！"

剑气如虹，斩在暗影豹身上。

暗影豹发出惨叫，被击退数步。

"现在的我，已经可以轻松击败元婴初期的魔兽了。"秦寒自信道。

他继续进攻，剑气纵横。

百招后，暗影豹被击杀。

"第一只。"秦寒取走暗影豹的晶核，继续前进。

接下来的几天，秦寒在暗影秘境中猎杀魔兽，收集材料。

他的战斗经验在不断提升，对元婴期的力量掌控也越来越熟练。

**剧情要点**：秦寒探索暗影秘境，遭遇元婴初期魔兽暗影豹，展现实力轻松击杀，继续深入秘境收集材料。

# 第一百五十四章 秘境收获

时间：探索秘境三天后
地点：暗影秘境深处
人物：秦寒

秦寒在暗影秘境中已经探索了三天，收获颇丰。

"元婴初期魔兽五只，元婴中期魔兽两只。"秦寒盘算着。

"还有各种珍贵材料：百年灵草十株，千年灵木一段，玄铁精矿一块..."

这些材料，价值至少十万上品灵石。

"差不多了。"秦寒满意地点头。

他准备返回宗门，提交任务。

不过，就在这时，他感受到一股强大的气息从秘境深处传来。

"是元婴后期的魔兽..."秦寒心中一惊。

他赶紧隐藏气息，小心翼翼地退出秘境。

"现在的我，还不足以对抗元婴后期的魔兽。"秦寒暗道。

离开暗影秘境后，秦寒返回玄天宗，提交任务。

"任务完成，这是奖励。"执事递给秦寒五万上品灵石。

"多谢。"

秦寒接过灵石，心中满意。

加上之前的积累，他现在有约八万上品灵石。

"这些资源，足够我冲击元婴中期了。"秦寒暗道。

# 第一百五十五章 宗门贡献

时间：返回宗门后
地点：玄天宗
人物：秦寒、宗门弟子

提交任务后，秦寒在宗门内走动。

一路上，很多弟子向他行礼。

"秦长老好！"
"秦长老，听说您突破元婴了，恭喜！"
"秦长老是我们玄天宗的骄傲！"

秦寒微笑着点头致意。

"突破元婴后，宗门对我的态度更尊敬了。"秦寒暗道。

他来到藏经阁，想要兑换一些功法。

藏经阁是宗门存放功法的地方，分为多层。秦寒现在可以进入第四层，兑换一些元婴期的功法。

"这位长老，请问需要兑换什么功法？"藏经阁执事问道。

"我想看看元婴期的功法。"秦寒说道。

"元婴期功法在这边。"执事带秦寒来到一个柜台。

"这里有《元婴诀》、《元神修炼法》、《分身术》等功法。每本需要一万贡献点。"

秦寒查看自己的贡献点。

"我有五万贡献点，可以兑换五本功法。"秦寒盘算着。

"我兑换《元婴诀》、《元神修炼法》、《分身术》、《炼体诀》、《神识修炼法》。"

"好的，长老。"执事递给秦寒五枚玉简。

"多谢。"

秦寒接过玉简，返回洞府。

"这些功法，对我提升实力很有帮助。"秦寒满意地点头。

他决定先闭关修炼，巩固元婴初期的修为，然后冲击元婴中期。

灵气如潮水般涌入体内，被元婴吸收。

元婴在丹田中缓缓旋转，散发出璀璨的金光。

每一次旋转，都会吞噬大量灵气，然后将其转化为纯粹的力量。

密室内的灵气浓度瞬间提升数倍。

修炼速度也大幅提升。

时间一天天过去，秦寒的修为在不断提升。

元婴越来越强大，灵力也越来越精纯。

终于，在两个月后，秦寒感受到了突破的契机。

"要突破了！"

# 第一百五十七章 突破元婴中期

时间：闭关两个月后
地点：密室
人物：秦寒

秦寒感受到突破的契机，全力冲击。

"破！"

他全力催动《元婴诀》和《苦海经》。

体内的元婴开始变化，原本三寸大小的元婴，开始生长。

四寸、五寸、六寸...

最终，元婴长到一尺大小，散发出更加强大的气息。

"元婴中期，突破成功！"

秦寒睁开眼睛，眼中闪过精芒。

他站起身，感受着元婴中期的力量。

神识范围扩大到方圆两百里，灵力精纯度再次提升，力量也大幅增强。

"现在的我，可以轻松击败元婴中期的修士。"秦寒自信道。

"甚至，与元婴后期也能一战。"

就在这时，天空中传来雷声。

轰隆隆！

"雷劫？"

秦寒心中一惊。

突破元婴中期，也会引来雷劫吗？

他赶紧催动黑铁灵甲，防御提升。

轰！

第一道雷劫落下，被灵甲挡住。

第二道、第三道...

一共七道雷劫，全部被秦寒挡住。

"元婴中期的雷劫，比元婴初期弱很多。"秦寒松了一口气。

不过，他知道，这只是开始。

"接下来的修炼，会更加艰难。"

秦寒返回密室，继续巩固修为。

# 第一百五十九章 洞天联盟会议
