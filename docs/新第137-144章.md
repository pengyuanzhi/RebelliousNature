# 第一百三十七章 发现父亲的旧伤

时间:父子深入交谈后的第二天
地点:玄天宗,长老洞府
人物:秦寒、父亲

第二天一早,秦寒来到父亲的洞府,发现父亲正在打坐修炼。

"父亲,您起得真早。"秦寒笑道。

父亲睁开眼睛,笑着说:"老了,睡不着。而且我想着,既然你回来了,我也该好好修炼了。"

"父亲说笑了。"秦寒说,"您还年轻,好好修炼,一定能有所成就。"

"希望如此吧。"父亲苦笑,"不过我的资质一般,恐怕很难有大的突破了。"

"父亲,不要这样说。"秦寒说,"修炼最重要的是坚持。只要不放弃,总会有进步的。"

父子二人聊了一会儿,秦寒突然注意到父亲的脸色有些苍白。

"父亲,您怎么了?"秦寒关切地问,"脸色不太好。"

"没事。"父亲摆手,"可能是昨晚修炼有些急了。"

秦寒心中一动,伸手搭在父亲的手腕上。体内的金丹之力缓缓流转,探查父亲的身体状况。

片刻后,秦寒的脸色变得凝重起来。

"父亲,您的旧伤..."秦寒沉声道。

父亲脸色一变:"你怎么知道我有旧伤?"

"我刚才给您检查了一下。"秦寒说,"您的丹田有裂痕,经脉也有损伤。这是旧伤,而且很严重。"

父亲沉默了片刻,叹了口气:"是的,我确实有旧伤。当年从上界回来时,受了重伤。这些年虽然养好了很多,但一直没有完全痊愈。"

"为什么不早告诉我?"秦寒有些生气。

"不想让你担心。"父亲说,"而且这是几十年前的旧伤了,很难治愈。"

"不难治愈。"秦寒坚定地说,"父亲,我在万宝大会上买了疗伤丹药,应该能治好您的旧伤。"

"真的?"父亲眼中闪过一丝希望。

"真的。"秦寒肯定地点头,"那是一种四阶丹药'复元丹',专门治疗旧伤。虽然很贵,但值得。"

"可是..."父亲有些犹豫,"四阶丹药价值连城,你花了多少钱?"

"父亲,您是我父亲,说什么钱不钱的。"秦寒说,"只要能治好您的旧伤,花多少钱都值得。"

父亲眼眶湿润:"寒儿,谢谢你..."

"父亲,您是我父亲,为什么要说谢谢?"秦寒笑了。

父亲也笑了,眼角却挂着泪花。

"好,那就试试吧。"父亲说。

秦寒取出"复元丹",这是一枚深紫色的丹药,散发着浓郁的药香。父亲接过丹药,深吸一口气,然后服下。

丹药入腹,顿时化作一股暖流,流遍全身。父亲的脸色逐渐红润起来。

秦寒伸出双手,按在父亲的背上。体内的金丹之力缓缓流出,进入父亲体内,帮助化解药力,驱除旧伤。

金丹后期的灵力,精纯而强大。在秦寒的帮助下,药力被迅速化解,源源不断地修复着父亲的旧伤。

这个过程持续了整整一个时辰。

一个时辰后,秦寒收回双手,脸上露出疲惫的神色。

"父亲,您的旧伤已经治好了七成。"秦寒说,"剩下的,需要慢慢调养。再吃两三枚丹药,应该就能完全痊愈了。"

父亲活动了一下身体,惊喜地发现,困扰自己几十年的旧伤,竟然真的好了大半!

"寒儿,真的...真的太感谢你了!"父亲激动地说。

"父亲,您是我父亲,说什么感谢。"秦寒笑了。

父亲紧紧抱住秦寒,眼泪流了下来。

# 第一百三十八章 父亲痊愈

时间:疗伤三天后
地点:玄天宗,长老洞府
人物:秦寒、父亲

三天后,秦寒又给父亲服用了两枚复元丹。

这一次的疗伤过程更加顺利。父亲的旧伤在丹药和秦寒灵力的帮助下,被完全治愈。

"寒儿,我的旧伤...完全好了!"父亲激动地说。

他感受了一下体内的情况,发现丹田的裂痕已经完全修复,经脉也恢复了顺畅。困扰自己几十年的旧伤,竟然真的痊愈了!

"希望如此吧。"父亲苦笑,"不过我已经老了,即使能突破,恐怕也到不了多高的境界了。"

"父亲,不要这样说。"秦寒说,"修炼最重要的是坚持。只要不放弃,总会有进步的。而且我这里有很多修炼资源,您需要什么,尽管跟我说。"

"好,有儿子这句话,我就放心了。"父亲笑道。

接下来的几天,秦寒一直陪伴父亲。父子二人经常聊天,有时候聊修炼,有时候聊生活,有时候聊过去。

"寒儿,你这些年,吃了不少苦吧?"父亲关切地问。

"还行。"秦寒笑道,"修炼之路本来就是苦修。这点苦,不算什么。"

"你这孩子,从小就很坚强。"父亲感叹道,"你母亲如果在天有灵,一定为你骄傲。"

提到母亲,父亲的眼神黯淡了下来。

"父亲,您能跟我多讲讲母亲的事吗?"秦寒问。这是他第一次主动询问母亲的事。

父亲沉默片刻,缓缓点头:"好。"

父亲泣不成声,再也说不下去了。

轰!

秦寒来到太华宗,再次见到了苏清歌。

"秦寒哥哥!"苏清歌看到他,眼中闪过惊喜。

"清歌,我来向你告别。"秦寒说。

"告别?你要去哪里?"苏清歌问。

"我要返回中界了。"秦寒说,"中界将发生大事,我必须回去。"

"哦..."苏清歌有些失落,"那你什么时候回来?"

"不确定。"秦寒说,"不过你放心,等我在中界稳定下来,一定来接你过去。"

"嗯,我等你。"苏清歌点头。

两人沉默了一会儿。

"清歌,你要继续努力修炼。"秦寒说,"争取早日突破到元婴期,这样我接你过去时,你也有自保之力。"

"好,我会的。"苏清歌说,"秦寒哥哥,你也要小心。中界肯定比下界危险,你一定要保护好自己。"

"放心吧。"秦寒笑道,"我现在可是金丹后期修士,一般的危险,我不怕。"

"嗯,但还是小心为好。"苏清歌还是不放心。

秦寒看着苏清歌,心中涌起一股暖意。

从认识她到现在,已经过去好几年了。当年的青州城相遇,仿佛还在昨天。那时的苏清歌,还是个柔弱的少女,需要他的保护。而现在的苏清歌,已经成长为金丹后期修士,能够独当一面了。

"清歌,等我。"秦寒认真地说。

"嗯,我等你。"苏清歌也认真地说。

两人相拥而别。

秦寒离开太华宗,飞向中界传送阵。

传送阵位于青州域最东端,是一座古老的阵法。据说,这是上古时期的大能留下来的,专门用来连接下界和中界。

秦寒来到传送阵前,深吸一口气。

"下界,我还会回来的。"秦寒在心里默默说道,"等我成为真正的强者,一定回来,保护我想保护的一切!"

他踏上传送阵,灵力注入其中。

轰!

传送阵启动,一道白光笼罩秦寒。下一刻,他的身影消失在原地,出现在另一个世界。

中界,他又回来了!
