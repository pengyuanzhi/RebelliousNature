# 第一百零八章 围剿后的总结

时间:围剿血煞教后第五天
地点:太华宗,主殿 → 秦寒洞府
人物:秦寒、太华宗主、诸位长老、苏清歌、赵灵儿

太华宗主殿内,秦寒站在大殿中央,向宗主和诸位长老汇报围剿血煞教的经过。

"...就是这样。"秦寒说道,"我们击杀了血煞教主,为下界除了一大害。"

"好!好啊!"宗主连声赞叹,"秦寒,你这次立下大功!不仅击杀了血煞教主,还缴获了大量血煞教的资源。这些资源,足够太华宗使用数十年!"

"宗主过奖了。"秦寒拱手,"这是弟子应该做的。"

"不,这不是过奖。"宗主认真地说,"秦寒,你以金丹后期修为,击杀金丹圆满的血煞教主,这是越级挑战!这样的战绩,足以载入太华宗的史册!"

诸位长老也纷纷点头,眼中满是赞赏。

"秦寒,你想要什么奖励?"宗主问道。

秦寒想了想,说道:"弟子想要一些修炼资源,帮助巩固修为。"

"好!"宗主点头,"宗门会给你提供充足的修炼资源。包括灵石、丹药、法宝等。而且,我还会给你一枚破境丹,帮助你将来突破元婴期。"

"多谢宗主!"秦寒拱手。

"不用客气。"宗主笑道,"你为宗门做出这么大贡献,这是你应得的。"

宗主取出一枚丹药,递给秦寒。这枚丹药散发着浓郁的药香,仅仅是拿在手中,就能感受到其中蕴含的强大药力。

"这就是破境丹。"宗主说,"是辅助突破境界的丹药,对于金丹期突破元婴期有很大帮助。以你现在的修为,还用不上。但等将来你达到金丹圆满,这枚丹药会派上大用场。"

秦寒郑重地收下破境丹。

"还有。"宗主又拿出一本秘籍,"这是我们太华宗的镇派功法之一——《元婴诀》。这部功法专门讲述如何突破元婴期,非常有价值。你将来突破元婴期时,可以参考。"

秦寒接过秘籍,翻开几页,发现里面详细记载了突破元婴期的方法和注意事项。

"多谢宗主!"秦寒再次拱手。

"不用客气。"宗主笑道,"你是我宗门的天才弟子,宗门当然要全力支持你。对了,你现在的修为是金丹后期,打算什么时候突破元婴期?"

秦寒沉思片刻,说道:"弟子打算先巩固金丹后期的修为,等达到金丹圆满后,再考虑突破元婴期。"

"明智的选择。"宗主点头,"金丹后期是金丹期的重要阶段,需要打好基础。只有根基扎实,将来突破元婴期才会顺利。"

"是的。"秦寒点头,"弟子也是这样想的。"

"好。"宗主道,"那你先回洞府修炼吧。有什么需要,尽管来找我。"

"是。"秦寒拱手,退出大殿。

回到洞府后,秦寒将破境丹和《元婴诀》收好。

"破境丹、《元婴诀》..."秦寒喃喃自语,"这些,都是为将来突破元婴期准备的。但现在,我还需要继续巩固金丹后期的修为。"

他盘膝坐下,开始修炼。

体内的金丹缓缓旋转,吸收着周围的灵气。金丹后期的修炼,比金丹初期和中期更加深奥。秦寒运转《苦海经》,这门功法越来越深奥。他发现自己对《苦海经》的理解还远远不够,这部功法蕴含的大道至理,需要慢慢领悟。

# 第一百零九章 返回玄天宗

时间:太华宗休整三天后
地点:太华宗 → 玄天宗
人物:秦寒、苏清歌、赵灵儿

三天后,秦寒向太华宗主辞行。

"秦寒,你要回玄天宗了?"宗主问。

"是的。"秦寒点头,"弟子已经在太华宗待了很长时间,该回去了。"

"好。"宗主道,"你回去后,好好修炼。太华宗永远欢迎你。"

"多谢宗主。"秦寒拱身。

秦寒带着苏清歌和赵灵儿,离开太华宗,飞向玄天宗。

一路上,三人有说有笑。

"秦寒哥哥,这次围剿血煞教,你立了大功呢!"苏清歌笑着说。

"都是大家的功劳。"秦寒谦虚地说,"如果不是苏清歌和灵儿帮忙,我也不可能成功。"

"我们也没做什么。"赵灵儿说,"真正击杀血煞教主的,是你。"

"呵呵,不要争了。"秦寒笑道,"我们是一个团队,功劳是大家的。"

很快,三人就来到了玄天宗。

玄天宗的弟子们看到秦寒回来,纷纷围了上来。

"秦师兄,你回来了!"

"秦师兄,听说你击杀了血煞教主,是真的吗?"

"秦师兄,你太厉害了!"

秦寒微笑着回应弟子们的问题。

很快,玄天宗宗主也知道了秦寒回来的消息,亲自出来迎接。

"秦寒,欢迎回来!"宗主笑道。

"宗主。"秦寒拱手。

"快进来,我有个好消息要告诉你。"宗主说。

三人来到主殿,宗主屏退左右,只留下秦寒、苏清歌和赵灵儿。

"秦寒,我刚刚接到中界的传讯。"宗主神秘地说,"洞天真人让你尽快返回中界。"

"洞天真人?"秦寒心中一动,"师父找我有什么事吗?"

"不太清楚。"宗主摇头,"但能肯定的是,一定是大事。洞真人很少主动找弟子,除非有重要的事情。"

秦寒沉思片刻,说道:"那我准备一下,过几天就返回中界。"

"不急。"宗主摆手,"你刚回来,先休息几天。而且,你可以在玄天宗再修炼一段时间,巩固一下金丹后期的修为。等准备充分了,再返回中界也不迟。"

"好。"秦寒点头。

"对了。"宗主说,"玄天宗决定晋升你为长老。"

"长老?"秦寒有些意外。

"是的。"宗主认真地说,"以你现在的修为和实力,完全有资格担任长老。而且,你为下界除掉了血煞教这个大害,这是巨大的贡献。"

"多谢宗主厚爱。"秦寒拱手。

"不用客气。"宗主笑道,"你是我宗门的天才弟子,这是你应得的。"

宗主取出一件长老法袍,递给秦寒:"这是长老法袍,穿上它,代表你是玄天宗的长老。"

秦寒接过法袍,郑重地穿上。

"多谢宗主。"秦寒再次拱身。

"不用客气。"宗主笑道,"对了,你现在是长老了,可以进入宗门的秘境修炼。秘境中的灵气浓度,比外界高数倍,非常适合修炼。"

"多谢宗主提醒。"秦寒说,"弟子会考虑的。"

从主殿出来后,秦寒、苏清歌和赵灵儿回到洞府。

"秦寒哥哥,你现在可是长老了!"苏清歌笑着说。

"没什么。"秦寒笑道,"只是一个称号罢了。最重要的是提升修为,保护我想保护的人。"

"嗯!"苏清歌和赵灵儿都点头。

# 第一百一十章 金丹后期的巩固

时间:返回玄天宗后
地点:玄天宗,秘境
人物:秦寒

秦寒决定进入秘境修炼,巩固金丹后期的修为。

玄天宗的秘境位于宗门后山,是一个独立的小空间。秘境中的灵气浓度,比外界高数倍,非常适合修炼。

秦寒进入秘境,找了一处幽静的山谷,盘膝坐下。

他闭上眼睛,开始修炼。

体内的金丹缓缓旋转,吸收着周围的灵气。秘境中的灵气非常浓郁,秦寒感觉到自己的修为在缓慢提升。

"金丹后期初期的修为,还需要继续巩固。"秦寒喃喃自语。

他运转《苦海经》,这门功法越来越深奥。他发现自己对《苦海经》的理解还远远不够,这部功法蕴含的大道至理,需要慢慢领悟。

"以苦炼心,以痛证道..."秦寒喃喃自语。

《苦海经》的核心,就是将所有的痛苦转化为力量。秦寒回想起自己经历的所有苦难——母亲去世、父亲受伤、婆婆牺牲、修炼艰辛...

这些痛苦,刻在他的心上,成为了永恒的伤疤。

但现在,这些伤疤不再是伤痛,而是力量!

每一次回想,秦寒都能感受到一股力量从心底涌出,这股力量让他的修为不断提升。

轰!

秦寒感觉到自己的修为又精进了不少。金丹后期初期的修为,正在向金丹后期中期迈进。

"这样的速度,真是骇人听闻。"秦寒苦笑,"不过,我需要的是稳定,而不是速度。太快的突破,容易根基不稳。"

他决定放慢速度,先巩固现有修为。

接下来的三个月,秦寒一直在秘境中闭关修炼。他没有急于突破,而是专心巩固金丹后期的修为。

每天,他都会花数个时辰修炼《苦海经》,感悟其中的大道至理。剩下的时间,他会练习剑法、身法、瞳术等战斗技能。

三个月后,秦寒的修为已经彻底稳固在金丹后期中期。

"是时候出关了。"秦寒喃喃自语。

他站起身,准备离开秘境。

# 第一百一十一章 苏清歌的提议

时间:秦寒出关后
地点:玄天宗,秦寒洞府
人物:秦寒、苏清歌

秦寒从秘境出来后,回到洞府,发现苏清歌正在等他。

"秦寒哥哥,你出关了!"苏清歌笑着说。

"嗯。"秦寒点头,"三个月的闭关,我的修为有所巩固。"

"哦?巩固到什么境界了?"苏清歌好奇地问。

"金丹后期中期。"秦寒说。

"什么?!"苏清歌震惊了,"金丹后期中期?这么快?"

"是的。"秦寒笑道。

"天呐..."苏清歌感叹道,"秦寒哥哥,你的修炼速度真是太快了!我到现在才刚刚达到金丹后期初期,你已经是中期了..."

"清歌,你也不慢。"秦寒安慰道,"你能达到金丹后期,已经非常了不起了。要知道,你可是天生废体啊。"

"这都是因为你。"苏清歌认真地说,"如果不是你帮我修复灵根,我根本不可能修炼到金丹期。"

"清歌,你过奖了。"秦寒笑道。

"不过..."苏清歌犹豫了一下,说道,"秦寒哥哥,我有个提议。"

"什么提议?"秦寒问。

"双修。"苏清歌认真地说,"我听说,双修可以让两人的修为都大幅提升。而且,双修还能增进两人的感情。"

秦寒愣了一下。

双修...他当然听说过。这是修真界中非常普遍的修炼方式。两个人通过阴阳调和,可以让修为都大幅提升。

但是,双修需要两人都有深厚的感情基础,而且必须是心甘情愿。否则,不仅无法提升修为,还可能会走火入魔。

"清歌,你...愿意和我双修?"秦寒有些不敢相信。

"我愿意。"苏清歌认真地点头,"秦寒哥哥,我早就把你当成我最重要的人了。我愿意和你一起修炼,一起变强,一起走到大道的尽头。"

秦寒看着苏清歌认真的眼神,心中涌起一股暖意。

"好。"秦寒握住苏清歌的手,"那我们就一起双修。"

苏清歌脸红了,但还是坚定地点头。

# 第一百一十二章 双修修炼

秦寒的阳刚灵力,进入苏清歌体内,帮助她温养经脉,强化根基。

两人的修为,都在缓缓提升。

这一夜的双修,不仅让两人的修为都大幅提升,更重要的是,让两人的感情更加深厚了。
