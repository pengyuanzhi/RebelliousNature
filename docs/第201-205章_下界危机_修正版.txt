# 第二百零一章 魔影再现

时间：返回太华宗一个月后
地点：下界·青州域·太华宗
人物：秦寒、苏清歌

秦寒返回太华宗已经一个月。

这一个月来,他除了陪伴苏清歌,就是闭关修炼,稳固元婴中期后期的修为。

之前的战斗中,他经历了生死磨砺,修为已经悄然突破。

元婴中期中期→元婴中期后期→元婴中期大圆满!

距离元婴后期,只有一步之遥。

"差不多了……"秦寒喃喃。

但就在这时,命轮眼再次示警!

这次示警比之前更加强烈!

"怎么回事?"秦寒震惊。

他全力运转命轮眼,看向幽州域的方向。

那里,一团巨大的黑色因果线在翻涌!

而且,这次的规模比之前更大!

"魔界……"秦寒咬牙。

他意识到,魔界并没有放弃入侵下界的计划。

之前的失败,只是暂时的。

魔界正在酝酿更大的入侵!

"必须做好准备。"秦寒立即出关。

他前往掌门大殿,向玄天真人汇报情况。

"什么?魔界还要入侵?"玄天真人大惊。

"是的。"秦寒点头,"而且这次规模会更大。我看到的因果线,比之前浓重十倍!"

"那怎么办?"一位长老忧心忡忡。

"备战。"秦寒坚定道,"我们必须做好最坏的打算。"

"我同意。"玄天真人点头,"立即通知其他宗门,我们再次备战!"

"是!"

联盟再次成立。

但这次,大家都明白,真正的考验才刚刚开始。

# 第二百零二章 元婴后期

时间：备战期间
地点：太华宗·闭关室
人物：秦寒

备战开始后,秦寒意识到,自己的实力还不够。

元婴中期大圆满虽然不错,但面对魔界的大规模入侵,还是有些勉强。

"我需要突破到元婴后期。"秦寒下定决心。

他取出所有积攒的资源,准备冲击元婴后期。

元婴后期,是元婴期的一个重要分水岭。

达到元婴后期,才能掌握真正的神通。

比如——神识化形、瞬移、领域……

这些都是元婴后期才能掌握的能力。

"开始吧。"秦寒盘膝坐下。

他全力运转《苦海经》,开始冲击元婴后期。

元婴在丹田中翻涌,不断吸收灵气。

渐渐的,元婴开始发生变化。

它变得更加凝实,更加清晰。

而且,秦寒感觉到自己的神识也在快速增强。

神识范围从方圆千里,扩展到方圆三千里!

"这就是元婴后期……"秦寒喃喃。

但就在这时,突破遇到了瓶颈。

"还差一点……"

秦寒思考着,自己还缺少什么。

突然,他想起了命轮眼。

"命轮眼能不能帮我突破?"

秦寒尝试着让命轮眼参与突破过程。

第四重——改变命运!

命轮眼的力量注入元婴,改变元婴的"命运"。

原本停滞的突破,再次启动!

轰!

元婴猛地一震,彻底突破!

元婴后期初期!

"成功了……"秦寒激动。

但随即,他感到一阵虚弱。

突破的消耗太大了。

"先稳固修为吧。"秦寒决定。

他开始巩固元婴后期初期的修为。

三个月后。

修为稳定在元婴后期中期。

"现在,我有实力保护下界了。"秦寒满意。

他走出闭关室,准备继续备战。

# 第二百零三章 联盟大会

时间：突破后
地点：下界·青州域·太华宗
人物：秦寒、各派掌门

秦寒出关后,立即召集联盟大会。

各派掌门齐聚太华宗,商议对策。

"诸位,"秦寒道,"魔界即将再次入侵。这次规模更大,我们必须做好充分准备。"

"秦道友说得对。"丹塔塔主点头,"我们丹塔愿意提供丹药支持。"

"我们玄天宗愿意提供阵法支持。"玄天宗宗主表态。

"我们药王谷愿意提供疗伤支持。"药王谷谷主也说。

"多谢诸位。"秦寒抱拳,"有了大家的支持,我们一定能击退魔界。"

"秦道友,"玄天真人道,"你现在的修为是?"

"元婴后期中期。"秦寒道。

"元婴后期!"众掌门震惊。

"恭喜!"丹塔塔主赞叹,"元婴后期,在下界已经是一方强者了。"

"不敢当。"秦寒谦虚,"面对魔界,我还是不够。"

"别谦虚了。"玄天真人笑道,"有你在,下界就有希望。"

"那我们继续商议对策吧。"秦寒道。

众人开始详细规划。

第一,建立侦查系统。

在各州域设立观察点,一旦发现异常,立即通报。

第二,建立快速反应机制。

一旦某处遇袭,其他州域能立即支援。

第三,建立资源调配系统。

丹药、法宝、灵石,统一调配,避免浪费。

第四,建立指挥系统。

由秦寒担任总指挥,统一调配联盟军队。

"这四个系统,能大大提升我们的防御能力。"秦寒道。

"同意。"众掌门齐声应道。

"那就开始吧。"

联盟开始全面运转。

侦查、反应、调配、指挥,四大系统逐步建立。

一个月后。

联盟已经完全运转起来。

"这样下去,下界的防御能力会大大提升。"苏清歌赞叹。

"是的。"秦寒点头,"但还要做得更好。"

他继续完善联盟的体系。

又过了一个月。

联盟运转非常顺畅。

下界进入战备状态。

"希望能撑过这次危机……"秦寒喃。

# 第二百零四章 魔界先锋

时间：备战三个月后
地点：下界·幽州域边缘
人物：秦寒、联盟先遣队

就在联盟备战如火如荼时。

一个消息传来。

幽州域边缘,出现了魔界生物!

"来了……"秦寒凝重。

他立即带领联盟先遣队,前往幽州域。

先遣队由秦寒亲自带队,共五百名精英修士。

其中,最强的有元婴初期,最弱的也是金丹后期。

"诸位,"秦寒道,"我们这次的任务是侦查,不要轻易交战。"

"明白!"众人齐声应道。

五百修士快速前进,半天后到达幽州域边缘。

远处,可以看到大量的魔界生物。

魔兵、魔将、甚至还有魔帅!

数量超过一万!

"这么多……"林玄脸色凝重。

"别慌。"秦寒冷静道,"先观察情况。"

他利用命轮眼,观察魔界生物。

第一重——看因果线!

在命轮眼的视野中,秦寒看到这些魔界生物身上,都有一条因果线,连接着虚空的某个地方。

"那是……魔界通道的位置!"秦寒恍然大悟。

他顺着因果线,找到了魔界通道的坐标。

"找到了。"秦寒道。

"在哪里?"白素素问。

"幽州域北部,荒漠深处。"秦寒道,"那里有一座传送阵,连接着魔界。"

"那我们怎么办?"慕容雪问。

"封锁。"秦寒道,"我们必须封锁传送阵,阻断魔界的增援。"

"可是,魔界生物太多了……"林玄担忧。

"别怕。"秦寒道,"我们有办法。"

他开始制定计划。

计划很简单:利用联盟的阵法优势,建立封锁线,将魔界生物困在荒漠中。

然后,逐步消灭。

"这个计划可行。"玄天宗的一位阵法师点头,"我们可以布下'九天十地绝灭阵',封锁方圆百里。"

"好。"秦寒道,"那就开始布阵。"

联盟修士开始布阵。

秦寒则吸引魔界生物的注意,为布阵争取时间。

"魔界杂碎,来战!"秦寒喝道。

他全力出手,元婴后期的气息爆发,震慑群魔。

"人类修士……杀!"

魔界生物被吸引,冲向秦寒。

秦寒且战且退,将魔界生物引入包围圈。

"差不多了。"秦寒喃。

就在这时,阵法启动!

"九天十地绝灭阵,起!"

无数阵纹浮现,将魔界生物困在其中。

"什么?!"魔帅大惊。

"晚了。"秦寒冷笑。

联盟修士齐出手,消灭被困的魔界生物。

经过一天的战斗。

一万魔界生物,全部被消灭!

"成功了!"众人欢呼。

但秦寒没有放松。

他盯着远处的荒漠深处,眼中闪过一丝担忧。

"真正的考验,还在后面……"

# 第二百零五章 魔界通道

时间：消灭先锋后
地点：幽州域·荒漠深处
人物：秦寒、联盟大军

消灭魔界先锋后,秦寒决定主动出击。

"我们不能被动防守。"秦寒道,"必须主动攻击魔界通道,从源头上解决问题。"

"可是,魔界通道在荒漠深处,那里肯定有强大的魔界生物把守。"林玄担忧。

"我知道。"秦寒点头,"但这是我们唯一的机会。"

"那我们进攻吧。"苏清歌坚定道。

"好。"秦寒点头。

联盟大军集结,共五万修士。

浩浩荡荡,开往荒漠深处。

三天后,他们到达了魔界通道的所在地。

一座巨大的传送阵,矗立在荒漠中央。

传送阵周围,有数百名魔将,还有十几位魔帅把守!

"这么多……"慕容雪凝重。

"别慌。"秦寒道,"我来对付魔帅,你们对付魔将。"

"明白!"众人齐声应道。

秦寒一马当先,冲向魔界通道。

"人类修士,找死!"一位魔帅怒喝。

"找死的是你!"秦寒冷哼。

他全力出手,元婴后期中期的实力全面爆发。

"苦海领域,开!"

方圆十里的领域展开,将十几位魔帅笼罩。

在苦海领域中,秦寒就是主宰!

"什么?!"魔帅们大惊。

他们发现自己的力量被压制了!

"该死……"

秦寒抓住机会,全力反击。

"苦海剑诀——横扫千军!"

恐怖的剑气横扫而出,瞬间斩杀数位魔帅。

剩余的魔帅吓坏了,纷纷逃跑。

"追!"秦寒道。

联盟修士齐追不舍,将剩余魔帅一一斩杀。

清理完魔帅后,秦寒将目光投向传送阵。

"破坏它!"秦寒下令。

联盟修士齐出手,攻击传送阵。

轰隆隆!

传送阵在密集的攻击下,开始崩塌。

"不……"虚空中传来一声怒吼。

传送阵彻底毁坏!

"成功了!"众人欢呼。

但就在这时,虚空中出现了一道裂缝。

裂缝中,走出一道身影。

身影全身笼罩在黑雾中,气息恐怖,达到了——元婴后期圆满!

"元婴后期圆满……"秦寒凝重。

"人类修士,"黑影开口,"你们毁了我的传送阵,该死!"

"你是谁?"秦寒冷冷道。

"本座乃是魔界先锋军统领,魔王麾下十大魔将之一,黑煞!"黑影道。

"黑煞……"秦寒眯眼。

"你们都要死!"黑煞一掌拍出。

恐怖的魔气如海啸般涌来。

"挡住!"秦寒全力抵挡。

他与黑煞展开激战。

两人的战斗,惊天动地。

周围的联盟修士纷纷后退,生怕被波及。

秦寒虽然也是元婴后期,但他只是中期,而黑煞是圆满。

实力差距明显。

"该死……"秦寒咬牙。

他准备燃烧精血,提升实力。

但就在这时——

"我来帮你!"

数道身影从远处飞来。

是丹塔塔主、玄天宗宗主、药王谷谷主……

下界所有的元婴期强者,都来了!

"诸位……"秦寒感动。

"我们一起。"丹塔塔主道。

下界总共有十二位元婴期。

加上秦寒,一共十三位。

"对付黑煞,我们联手!"丹塔塔主道。

"明白!"众人齐声应道。

十三位元婴期,分别施展最强的神通,轰向黑煞。

黑煞在十三位强者的围攻下,渐渐落了下风。

"该死的人类……"黑煞愤怒。

他燃烧魔气,实力暴涨。

元婴圆满后期!甚至接近化神期!

"完了……"秦寒脸色难看。

"别怕。"丹塔塔主道,"我们也燃烧寿元!"

十三位强者同时燃烧寿元,实力暴涨。

"现在,我们拼了!"

众人全力出手,轰向黑煞。

黑煞在十三位强者的围攻下,终于不敌。

"本座还会回来的!"

他留下狠话,撤回虚空。

"追不追?"林玄问。

"不追。"秦寒摇头,"虚空那边是魔界,我们追不进去。"

"那怎么办?"

"封印裂缝。"秦寒道。

十三位强者联手,封印了虚空裂缝。

"成功了。"秦寒松了口气。

但代价很大。

联盟损失了数千名修士。

而且,十三位元婴期强者都燃烧了寿元。

"值得吗?"秦寒问。

"值得。"丹塔塔主道,"为了下界,百年寿元算什么。"

秦寒敬佩地看着这些老者。

"我会记住大家的恩情。"

"应该的。"丹塔塔主微笑。

众人返回联盟营地。

这一战,虽然惨胜,但下界再次击退了魔界的入侵。

"希望能有一段和平时期……"秦寒喃。
