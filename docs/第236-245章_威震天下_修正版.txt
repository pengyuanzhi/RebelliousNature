# 第二百三十六章 和平年代

时间：孩子们历练返回后
地点：下界·各州域
人物：秦寒、苏清歌

孩子们历练返回后，下界继续保持着和平。

秦寒继续完善联盟的体系，提升下界的实力。

一年，两年，三年……

转眼，十年过去了。

下界在和平中慢慢恢复，变得更加强大。

联盟的体系越来越完善，防御能力越来越强。

新增的元婴期强者，已经达到三十位！

"做得很好。"玄天真人赞叹，"下界能有今天，全靠你。"

"不。"秦寒摇头，"是大家一起努力的结果。"

"你太谦虚了。"玄天真人道。

秦寒笑了笑，没有再争辩。

他站在太华宗的山顶，俯瞰整个下界。

"这些年，下界变化很大……"他喃喃。

"是的。"苏清歌走到他身边，"比以前强大了很多。"

"但还不够。"秦寒道，"异界的威胁，随时可能降临。"

"你已经做得很好了。"苏清歌握住他的手，"不要太勉强自己。"

"我会注意的。"秦寒微笑。

这十年，秦寒的修为也稳定在元婴圆满大圆满。

虽然他没有急于突破化神期，但他一直在为突破做准备。

参悟法则、稳固道心、积累资源……

一切都准备得差不多了。

"也许，该考虑突破了。"秦寒暗道。

但他还是没有着急。

因为他知道，一旦突破化神期，就会引来更大的关注和挑战。

"再等等吧。"他决定。

# 第二百三十七章 天降异象

时间：和平十年后
地点：太华宗·山顶
人物：秦寒、苏清歌

这日，秦寒正在山顶观景。

突然，天空中出现了异象！

无数道光芒从天而降，笼罩整个下界！

"这是什么？"秦寒凝重。

他全力运转命轮眼，观察异象的来源。

第一重——看因果线！

在命轮眼的视野中，秦寒看到了恐怖的一幕。

虚空中，出现了三座巨大的传送阵！

传送阵中，涌出了无数异界生物！

"异界又来了……"秦寒咬牙。

这次的规模，比之前任何一次都大！

"必须立即备战！"

秦寒召集联盟成员，召开紧急会议。

"诸位，"秦寒道，"异界发动了全面入侵。这次规模空前，我们有麻烦了。"

"有多少？"丹塔塔主问。

"超过百万。"秦寒道，"而且，还有化神期强者坐镇。"

"百万……"众掌门脸色惨白。

"别怕。"秦寒道，"只要我们团结一致，就有机会。"

"你说，我们听你的。"玄天真人道。

"好。"秦寒点头，"第一，立即集结所有兵力。第二，开启所有防御阵法。第三，向中界求援。第四，做好最坏的打算。"

"明白！"众掌门齐声应道。

整个下界，再次进入战备状态。

这是一场前所未有的浩劫。

而下界能否度过，就看这场大战了。

# 第二百三十八章 百万魔军

时间：异界入侵后
地点：下界·边缘地带
人物：秦寒、联盟大军

三座传送阵，源源不断地涌出异界生物。

百万异界大军，如潮水般涌入下界。

"挡住！"秦寒喝道。

联盟大军集结，共十万修士。

十万对一百万，数量悬殊。

但联盟修士没有退缩。

"为了下界，杀！"

战斗开始。

这是一场前所未有的惨烈战斗。

异界生物源源不断，杀之不尽。

联盟修士一个接一个倒下，但没有人逃跑。

"援军还没到吗？"林玄焦急。

"快了。"秦寒咬牙。

他全力出手，元婴圆满大圆满的实力全面爆发。

"苦海领域——最大范围！"

方圆百里的领域展开，将数万异界生物笼罩。

在苦海领域中，秦寒就是主宰！

"杀！"

秦寒一剑斩出，数千异界生物当场毙命！

但异界生物太多了，根本杀不完。

"该死……"秦寒咬牙。

就在这时，异界联军的后方，出现了三道巨大的身影。

那是异界的统帅！

每一位都是——化神期！

"化神期……"秦寒脸色难看。

现在的他，只是元婴圆满，根本不是化神期的对手。

"拼了。"

秦寒准备燃烧寿元，换取短暂的力量。

但就在这时——

"我来帮你！"

虚空中，传来一道声音。

紧接着，数道身影从天而降！

是中界援军！

# 第二百三十九章 中界援军

时间：激战时
地点：下界·边缘地带
人物：秦寒、中界援军、异界联军

"秦道友，我们来晚了。"玄机道。

"不晚。"秦寒松了口气，"来得正是时候。"

中界援军有五十位化神期！

加上中界的百万大军，总兵力达到百万！

"反击！"玄机下令。

中界大军与下界联军会合，总兵力达到百万！

其中包括五十多位化神期，和秦寒这位元婴圆满！

"杀！"

百万大军，冲向异界联军。

异界统帅看到中界援军，脸色一变。

"中界竟然派这么多化神期……"

"撤！"

异界统帅准备撤退。

"别想走！"秦寒喝道。

他全力出手，施展最强神通。

"苦海剑诀——天地斩！"

恐怖的剑气斩向异界统帅。

异界统帅一掌挡住，但也被击退！

"该死……"

异界统帅不再犹豫，带着残军撤回传送阵。

"追！"玄机下令。

百万大军追击异界联军，一路斩杀无数。

最终，异界联军损失惨重，逃回异界。

"我们赢了！"下界修士欢呼。

"多谢中界援军。"秦寒向玄机行礼。

"应该的。"玄机微笑，"不过，这次虽然击退了异界，但它们还会来的。"

"我知道。"秦寒点头，"我们会做好准备。"

中界援军离开，秦寒开始清理战场。

这一战，下界损失惨重。

十万修士，只剩下两万。

"下界……还能撑吗……"秦寒喃。

# 第二百四十章 重建下界

时间：击退异界后
地点：下界·各州域
人物：秦寒、联盟残军

击退异界后，下界再次陷入困境。

十万修士，只剩两万。

而且，大部分都是带伤的。

"必须尽快恢复。"秦寒道。

他开始实施重建计划。

第一，救治伤员。

苏清歌带领丹塔的医师，全力救治伤员。

第二，补充兵员。

各派开始招收新弟子，补充兵力。

第三，修复阵法。

各州域的防御阵法，在战斗中受损严重，需要修复。

第四，调配资源。

统一调配下界的资源，优先保障前线。

"这些措施，能让下界尽快恢复。"秦寒道。

一年后。

下界的情况有所好转。

伤员基本痊愈，新兵开始补充，阵法修复完毕。

"还要继续努力。"秦寒道。

他继续推动重建计划。

又过了一年。

下界基本恢复。

联盟的兵力，恢复到五万。

虽然不如之前，但已经能够自保。

"差不多了。"秦寒满意。

他站在太华宗的山顶，俯瞰整个下界。

"经历了这么多磨难，下界依然挺过来了。"他喃。

"是的。"苏清歌走到他身边，"下界的人民，很坚强。"

"我们会继续守护下界。"秦寒握住她的手。

"嗯。"

两人相拥而立，看着远处的夕阳。

夕阳的余晖洒在下界的大地上，一切都显得那么宁静。

"希望能有一段真正的和平时期……"秦寒喃。

# 第二百四十一章 孩子成家

时间：重建完成后
地点：太华宗
人物：秦寒、苏清歌、秦天、秦月

重建完成后，秦寒开始考虑孩子们的终身大事。

"天儿已经三十岁了，该考虑成家了。"秦寒道。

"是的。"苏清歌道，"月儿也二十八岁了。"

"你说，给他们找个什么样的伴侣？"秦寒问。

"要看他们自己的意愿。"苏清歌道，"我们不能包办。"

"你说得对。"秦寒点头。

他找到秦天和秦月，谈这件事。

"天儿，月儿，"秦寒道，"你们也不小了，该考虑成家了。"

"成家？"秦天一愣，"我还想修炼……"

"修炼和成家不冲突。"秦寒道，"你看，我和你妈妈不也过得很好吗？"

"这……"秦天犹豫。

"月儿，你呢？"秦寒问。

"我听爸爸的。"秦月道。

"不用听我的。"秦寒笑道，"这是你们自己的大事，要自己决定。"

"那……"秦月想了想，"爸爸可以帮我留意。"

"好。"秦寒点头。

接下来的日子，秦寒开始留意合适的人选。

秦天最后看上了药王谷的一位女弟子。

两人相处一段时间后，情投意合，结为道侣。

秦月则看上了玄天宗的一位男弟子。

两人也最终结为道侣。

"很好。"秦寒欣慰，"孩子们都成家了。"

"是的。"苏清歌道，"我们也可以放心了。"

"嗯。"

一家人其乐融融。

# 第二百四十二章 孙辈出世

"曾祖父……"秦寒抱着重孙子，心中充满喜悦。

# 第二百四十四章 回归修炼

人物：秦寒

享受了一段时间的天伦之乐后，秦寒决定重新投入修炼。

"我不能一直陪伴家人。"他暗道，"我还需要继续提升实力，保护他们。"

他进入闭关室，开始修炼。

虽然是元婴圆满大圆满，但他还没有突破化神期。

"是时候突破了。"秦寒道。

他开始做最后的准备。

他之前已经初步参悟了空间法则，现在要更加深入。

但他突然停住了。
