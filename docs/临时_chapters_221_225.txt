# 第二百二十一章 讨伐魔道

时间：筹备完成后
地点：东域→北域（魔道联盟总部）
人物：秦寒、三方联盟大军、魔道联盟

三方联盟大军，浩浩荡荡朝北域进发。

北域是魔道联盟的大本营，位于天界北部。

联盟大军超过百万！

其中，金仙境强者上千，人仙境强者数百！

甚至还有几位太乙境的大能！

"这规模……"苏清歌惊叹。

"魔道联盟撑不住的。"秦寒道。

确实，魔道联盟虽然有实力，但面对三方联盟的围攻，根本没有胜算。

一个月后。

联盟大军到达北域边境。

魔道联盟早已集结兵力，准备迎战。

"他们来了……"魔道联盟盟主——一位太乙境初期的魔修，脸色凝重。

"盟主，我们该怎么办？"手下问。

"守！"魔盟主道，"死守总部，等待援军。"

"援军？"

"没错。"魔盟主道，"我们已经向魔族老家求援。只要援军赶到，我们就能反攻！"

"明白了。"手下点头。

战斗开始。

三方联盟如潮水般涌来，攻向魔道联盟的防线。

"杀！"

联盟修士齐声怒吼，气势如虹。

魔道联盟的修士拼死抵抗，但渐渐落了下风。

三天后。

魔道联盟的第一道防线被攻破！

"撤！"魔盟主下令。

魔道联盟撤退到第二道防线。

联盟大军继续进攻。

又过了五天。

第二道防线也被攻破！

魔道联盟退守总部——魔天城。

魔天城是魔道联盟的核心城池，防御极其强大。

"进城！"秦寒下令。

联盟大军包围魔天城，开始攻城。

攻城持续了整整十天。

魔天城的防御虽然强大，但挡不住百万大军的攻击。

终于，城墙被轰开！

"冲进去！"

联盟大军冲进魔天城。

魔道联盟的修士拼死抵抗，但依然无法阻挡。

"该死……"魔盟主咬牙。

他意识到，魔道联盟完了。

"撤！撤往魔族老家！"

魔盟主带领残部，逃往魔族老家。

"追！"秦寒下令。

联盟大军追击，一路追到北域边缘。

那里，有一道巨大的空间裂缝。

裂缝通向魔族老家——魔天魔域！

"他们逃进魔天魔域了。"玄明道。

"不能让他们逃了。"秦寒道。

他准备追进魔天魔域。

"少盟主，不可！"玄明劝道，"魔天魔域是魔族的老家，极其危险。"

"我知道。"秦寒道，"但必须斩草除根。"

"那让我陪你一起去。"苏清歌道。

"好。"秦寒点头。

两人带着精锐小队，追进魔天魔域。

魔天魔域的景象，与天界截然不同。

天空是暗红色的，大地是黑褐色的。

空气中弥漫着浓烈的魔气。

"和之前的魔界很像……"苏清歌道。

"没错。"秦寒道，"魔天魔域应该就是之前那个魔界的残存势力。"

"那我们更要小心。"苏清歌道。

两人小心翼翼地前进。

沿途，他们遇到了不少魔族。

但在精锐小队的面前，这些魔族都不堪一击。

三天后。

两人到达了魔天魔域的核心——魔神殿。

魔神殿是魔天魔域最重要的建筑，也是魔族领袖的居所。

"进去吧。"秦寒道。

两人进入魔神殿。

殿中，站着一位恐怖的魔族。

他的气息，竟然达到了太乙境后期！

"太乙境后期……"秦寒凝重。

"你们追到这儿来了？"魔族领袖冷笑。

"没错。"秦寒道，"今日就是你的死期。"

"狂妄！"魔族领袖怒了。

他全力出手，恐怖的魔气如山崩海啸。

秦寒用命轮眼预判，轻松躲避。

"苦海命运剑——斩！"

剑光落下，在魔族领袖身上留下伤痕。

"你……"魔族领袖大惊。

"人仙境中期，竟然能伤到我？"

"我还有帮手。"秦寒道。

苏清歌和精锐小队一起出手，围攻魔族领袖。

魔族领袖虽然强大，但面对众人的围攻，渐渐落了下风。

"该死……"魔族领袖咬牙。

他意识到，自己可能不是众人的对手。

"魔神附体！"

魔族领袖燃烧精血，实力暴涨。

太乙境后期圆满！

"该死……"秦寒脸色凝重。

太乙境后期圆满，太强了。

"只能用那招了。"

秦寒全力运转命轮眼。

第六重——改写天地规则！

第七重——逆转生死！

双管齐下！

在命轮眼的力量下，魔族领袖的规则被改写，生命被逆转。

"什么？！"魔族领袖大惊。

他发现自己的生命在飞速流逝！

"不可能！"

"没什么不可能。"秦寒冷声道。

"苦海命运剑——灭世斩！"

秦寒全力一剑，劈向魔族领袖。

"不——！"

魔族领袖发出一声绝望的惨叫。

剑光落下，将他劈成两半！

太乙境后期的魔族领袖，被斩杀！

"成功了……"苏清歌惊喜。

"魔天魔域的威胁，彻底消除了。"秦寒道。

两人带领精锐小队，返回天界。

消息传开，整个天界都沸腾了。

"少盟主太强了！"

"斩杀太乙境后期魔族领袖！"

"天界有救了！"

赞美之声此起彼伏。

秦寒和苏清歌返回正道城，受到热烈的欢迎。

玄天亲自出城迎接。

"秦寒，"玄天道，"你为天界立下了不朽的功勋。"

"盟主过奖了。"秦寒道。

"不是过奖。"玄天摇头，"从今日起，你就是天界的英雄。"

"多谢盟主。"秦寒抱拳。

接下来的日子，秦寒开始享受难得的和平时光。

他陪伴苏清歌，教导弟子，处理联盟事务。

一切都往好的方向发展。

但秦寒知道，这只是暂时的。

他的目标，是飞升到更高的世界——仙界！

要飞升仙界，必须达到大罗金仙境。

大罗金仙，是太乙境之上的境界。

也是天界最高的境界。

"我还有很长的路要走……"秦寒喃喃。

但他不着急。

他有足够的时间，慢慢修炼。

而且，他有苏清歌陪伴，有正道联盟支持。

一切都好。

# 第二百二十二章 第三卷小结

时间：讨伐魔道后
地点：东域·正道城
人物：秦寒、苏清歌、玄天

讨伐魔道联盟后，天界迎来了久违的和平。

秦寒作为正道联盟少盟主，开始总结这一路的经历。

从下界飞升天界，到加入正道联盟，再到讨伐魔道联盟。

短短几年，秦寒经历了太多。

"清歌，"秦寒道，"我们来总结一下吧。"

"总结什么？"苏清歌问。

"我们这些年的经历。"秦寒道。

两人开始回忆：

从下界开始，秦寒只是一个山坳少年。

母亲早逝，父亲为梦想牺牲。

婆婆教导他识字读书。

太华宗招徒，他加入外门。

大比第一，寻找父亲遗骸。

金丹大成，洞天大比夺魁。

幽州域调查魔界，发现魔界阴谋。

太华宗之战，击退魔界入侵。

建立下界联盟，联合正道、魔道、散修。

魔界二次入侵，秦寒突破化神期。

封印空间裂缝，切断魔界通道。

进入魔界，斩杀魔主，摧毁魔界。

返虚期，合道期，渡劫期。

飞升天界。

加入正道联盟，斩杀无数魔族。

担任少盟主，讨伐魔道联盟。

"我们经历了很多……"苏清歌感叹。

"没错。"秦寒点头，"但我们一路走来，也收获了很多。"

"你收获了什么？"苏清歌问。

"收获了力量。"秦寒道，"从凡人到渡劫期，再到人仙境。"

"收获了友情。"秦寒道，"林玄、白素素、慕容雪……还有很多朋友。"

"收获了爱情。"秦寒握住苏清歌的手，"最幸运的，是遇到了你。"

苏清歌脸红，微笑道："我也是。"

"我们还收获了责任。"秦寒道，"守护下界、守护天界的责任。"

"是的。"苏清歌点头。

"未来的路还很长。"秦寒道，"我们要继续前进。"

"飞升仙界，成为大罗金仙。"

"我陪你一起。"苏清歌道。

"谢谢。"秦寒微笑。

次日。

秦寒召集正道联盟的高层，召开会议。

"诸位，"秦寒道，"这些年来，我们一起经历了太多。"

"魔界入侵，魔族骚扰，我们都一一克服了。"

"现在，天界迎来了和平。"

"但和平是短暂的。"秦寒道，"我们不能懈怠。"

"未来，我们还要面对更多的挑战。"

"所以我提议，建立更完善的防御体系。"

"培养更多的强者。"

"储备更多的资源。"

"只有这样，我们才能应对未来的危机。"

"少盟主说得对！"众人齐声道。

"那我们就这样决定。"秦寒道。

会议结束后，秦寒开始实施计划。

建立防御体系。

培养强者。

储备资源。

一切都在有条不紊地进行。

三个月后。

秦寒收到一个消息。

下界传来消息，苏清歌的家族找到了！

"什么？！"苏清歌惊喜。

"你的家族？"秦寒问。

"没错。"苏清歌道，"我父母在我很小的时候就失踪了。我一直想知道他们去了哪里。"

"现在有消息了？"

"是的。"苏清歌点头，"据说他们被困在一处秘境中。"

"那我们去救他们。"秦寒道。

"好。"苏清歌感激。

两人准备返回下界，寻找苏清歌的父母。

一场新的冒险，即将开始。

# 第二百二十三章 返回下界

时间：决定返回下界后
地点：天界→下界
人物：秦寒、苏清歌

秦寒和苏清歌向玄天请辞。

"盟主，我们准备返回下界一趟。"

"返回下界？"玄天一愣，"为何？"

"清歌的父母找到了。"秦寒道，"被困在一处秘境中。我们想去救他们。"

"原来如此。"玄天点头，"应该去。家人团聚，是大事。"

"多谢盟主理解。"秦寒道。

"需要我派兵协助吗？"玄天道。

"不用。"秦寒道，"我们自己去就行。"

"好。"玄天道，"那你们一路小心。"

"会的。"

两人离开正道城，前往天界与下界的通道。

飞升者的通道，只能单向通过。

要从天界返回下界，需要走特殊的通道。

"时空通道。"秦寒道。

时空通道是天界和下界之间的特殊通道。

只有达到一定境界的修士，才能通行。

两人进入时空通道。

通道中，时间和空间都扭曲了。

秦寒和苏清歌感觉自己在不断坠落。

不知过了多久。

终于，两人出现在下界。

"下界的灵气，果然稀薄……"苏清歌感叹。

"因为我们习惯了天界的灵气。"秦寒道。

两人施展身法，朝秘境的位置飞去。

秘境位于青州域深处的一片山脉。

两人到达山脉，开始寻找秘境入口。

"在哪里呢……"苏清歌四处观察。

"命轮眼。"秦寒道。

他全力运转命轮眼，观察周围的一切。

在命轮眼的视野中，他看到了一些不寻常的能量波动。

"在那里。"秦寒指向一处山壁。

"好。"苏清歌道。

两人走到山壁前，秦寒一掌拍出。

"轰！"

山壁被轰开，露出了一个洞口。

"找到了。"秦寒道。

两人进入洞穴。

洞穴内部很深，两人走了很久才到达底部。

底部，是一扇巨大的石门。

石门上刻满了符文。

"这是封印……"秦寒道。

"能解开吗？"苏清歌问。

"应该可以。"秦寒道。

他全力运转命轮眼，观察符文的规律。

片刻后，秦寒找到了关键点。

"解开。"

秦寒一指点在关键点上。

符文开始消散，石门缓缓打开。

"成功了！"苏清歌惊喜。

两人进入石门。

石门后，是一个巨大的空间。

空间中央，有两个人影。

是一男一女，气息衰弱，显然被困了很久。

"父亲！母亲！"苏清歌冲过去。

"你是……"男子虚弱地睁开眼。

"我是清歌啊！"苏清歌流泪道。

"清歌……"女子也睁开了眼，"你长这么大了……"

"父亲，母亲，你们怎么了？"苏清歌问。

"我们被困在这里……已经五十年了……"男子道。

"五十年？！"苏清歌震惊。

"没错。"女子道，"当年我们来探索秘境，结果被困在这里。"

"好在你们的血脉之力支撑我们活到现在。"

"我带你们出去。"秦寒道。

他全力运转命轮眼，寻找破开秘境的方法。

片刻后，秦寒找到了。

"跟我来。"

秦寒带着苏清歌和她的父母，离开了秘境。

回到外面，苏清歌的父母终于呼吸到了新鲜的空气。

"终于出来了……"两人感慨。

"父亲，母亲，这是秦寒，我的道侣。"苏清歌介绍。

"秦寒……"男子打量秦寒，"人仙境中期，不错。"

"多谢伯父。"秦寒抱拳。

"你救了我们，我该谢谢你才对。"男子道。

"应该的。"秦寒微笑。

"对了，父亲，母亲，"苏清歌问，"你们当年为什么来探索秘境？"

"为了找一件宝物。"男子道，"听说这秘境中藏有一件上古法宝。"

"找到了吗？"

"找到了。"女子道，"就在秘境中。"

"我们可以去取吗？"秦寒问。

"可以。"男子点头。

"那我们去吧。"苏清歌道。

四人再次进入秘境，寻找那件上古法宝。

# 第二百二十四章 上古法宝

时间：救出苏清歌父母后
地点：秘境深处
人物：秦寒、苏清歌、苏父苏母

四人深入秘境，寻找上古法宝。

秘境中充满了危险的机关和强大的灵兽。

但对秦寒来说，这些都不算什么。

他轻松破解机关，斩杀灵兽，带领众人前进。

"好强……"苏父赞叹。

"清歌，你找了一个好夫婿。"

"父亲……"苏清歌脸红。

"对了，父亲，"苏清歌问，"这件法宝叫什么？"

"时空轮。"苏父道，"传说中的时空法宝。"

"时空轮？"秦寒眼睛一亮。

"没错。"苏母道，"据说这件法宝蕴含时空法则，可以帮助修炼者领悟时空法则。"

"那一定要找到。"秦寒道。

他对时空法则很感兴趣。

因为他的命轮眼，就与时空法则有关。

众人继续前进。

又走了一天。

终于，他们到达了秘境的最深处。

那里，有一个巨大的祭坛。

祭坛中央，悬浮着一个轮盘状的法宝。

轮盘上刻满了符文，散发着时空波动。

"那就是时空轮……"苏父道。

"太好了。"苏清歌道。

秦寒飞向祭坛，准备取走时空轮。

但就在这时——

一道黑影突然出现，挡在秦寒面前。

"法宝是我的！"黑影喝道。

秦寒仔细一看。

黑影是一位魔修！

魔修的气息，竟然达到了人仙境后期！

"人仙境后期……"秦寒凝重。

"你是谁？"秦寒问。

"本座乃魔道联盟的使者，魔煞。"魔修冷笑，"这件法宝，本座要了。"

"做梦。"秦寒冷声道。

"那就死！"魔煞全力出手。

恐怖的魔气如潮水般涌来。

秦寒用命轮眼预判，轻松躲避。

"苦海命运剑——斩！"

剑光落下，在魔煞身上留下伤痕。

"你……"魔煞大惊。

"人仙境中期，竟然能伤到我？"

"你太弱了。"秦寒道。

他继续攻击，剑光连连。

魔煞虽然强大，但在秦寒的攻势下，渐渐落了下风。

"该死……"魔煞咬牙。

"万魔噬天！"

魔煞全力出手，召唤出无数魔影。

秦寒不慌不忙。

"小世界，吞！"

秦寒全力运转小世界，将所有魔影吞噬。

"什么？！"魔煞大惊。

"结束了。"秦寒道。

"苦海命运剑——灭世斩！"

剑光落下，魔煞被斩成重伤！

"我认输！"魔煞急忙道。

"滚。"秦寒冷喝。

魔煞吓得落荒而逃。

"厉害。"苏父赞叹。

"过奖了。"秦寒道。

他取下时空轮，仔细观察。

轮盘上刻满了时空符文，蕴含着强大的时空法则。

"好东西。"秦寒满意。

"秦寒，这件法宝给你。"苏父道。

"给我？"秦寒一愣。

"没错。"苏父点头，"是你救了我们，这件法宝应该归你。"

"可是……"

"不用可是。"苏母微笑，"我们年纪大了，用不上这件法宝。你年轻，正好用得上。"

"那多谢伯父伯母。"秦寒抱拳。

他收起时空轮，准备参悟。

"我们回去吧。"秦寒道。

"好。"众人点头。

四人离开秘境，返回太华宗。

到达太华宗后，苏父苏母受到了热情的接待。

玄天真人亲自出来迎接。

"苏兄，苏嫂，"玄天道，"你们终于回来了。"

"多谢玄天兄照顾清歌。"苏父道。

"应该的。"玄天微笑。

接下来的日子，秦寒开始参悟时空轮。

时空轮蕴含的时空法则，深奥无比。

但秦寒有命轮眼，领悟起来快得多。

一个月后。

秦寒对时空法则的领悟，达到了新的高度。

人仙境中期圆满！

"差不多了。"秦寒满意。

他出关，准备返回天界。

"父亲，母亲，"苏清歌道，"我们要回天界了。"

"去吧。"苏父道，"天界才是你们的舞台。"

"我们会经常回来看你们的。"苏清歌道。

"好。"苏母微笑。

秦寒和苏清歌告别父母，通过时空通道返回天界。

回到天界后，秦寒继续担任少盟主。

同时，他也开始准备冲击更高的境界。

人仙境后期！

太乙境！

大罗金仙！

每一步，都是巨大的挑战。

但秦寒有信心。

他一定能达到。

# 第二百二十五章 第九章总结

时间：返回天界后
地点：东域·正道城
人物：秦寒、苏清歌

秦寒返回天界后，开始总结这一段经历。

从飞升天界，到加入正道联盟，再到讨伐魔道联盟，最后返回下界救苏清歌的父母。

这一路走来，收获满满。

**修为变化**：
- 下界：金丹期 → 元婴期 → 化神期 → 返虚期 → 合道期 → 渡劫期
- 天界：渡劫期 → 人仙境初期 → 人仙境中期 → 人仙境中期圆满

**重要事件**：
1. 飞升天界，到达青云城
2. 加入正道联盟，成为长老
3. 完成第一个任务：清剿魔修据点
4. 魔族第一次入侵，斩杀三位人仙境魔将
5. 第二次入侵，斩杀金仙境圆满大将军
6. 参加天界大会，击败魔道联盟代表
7. 进入苍穹秘境，获得龙血草等宝物
8. 魔族第三次入侵，斩杀人仙境后期大元帅
9. 接受少盟主身份，开始管理正道联盟
10. 讨伐魔道联盟，攻入魔天魔域
11. 斩杀太乙境后期魔族领袖，消除魔天魔域威胁
12. 返回下界，救出苏清歌父母
13. 获得上古法宝时空轮

**获得宝物**：
1. 返虚丹
2. 斩天剑（圣阶法宝）
3. 时空石
4. 《大日如来咒》（神阶功法）
5. 龙血草
6. 时空轮（上古法宝）

**重要人物**：
- 苏清歌：道侣，陪伴秦寒一路走来
- 玄天：正道联盟盟主，太乙境大能
- 玄明：正道联盟金仙后期修士
- 苏父苏母：苏清歌的父母，被困秘境五十年

秦寒的实力已经达到人仙境中期圆满，距离太乙境只有一步之遥。

而且，他获得了时空轮这件上古法宝，对时空法则的领悟更深了。

接下来，秦寒将继续冲击更高的境界。

同时，天界也将迎来新的挑战。

神秘势力即将出现，更大的阴谋正在酝酿。

秦寒能否应对？
