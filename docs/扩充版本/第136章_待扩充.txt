# 第一百三十六章 万宝大会开启

时间:万宝大会第一天
地点:万宝城
人物:秦寒、洞天真人、慕容雪、白羽、无数修士

万宝城,中界最繁华的城市,也是万宝大会的举办地。

这一天,无数修士从四面八方赶来。有金丹期的散修,有元婴期的宗门长老,甚至还有化神期的绝世强者。整个城市人山人海,热闹非凡。

"这就是万宝城吗?"秦寒站在城门口,被眼前的景象震撼了。

城墙高达千丈,通体由星辰铁打造,闪烁着金属光泽。城门上方,悬挂着"万宝城"三个大字,每一字都蕴含着恐怖的威压。

"走吧。"洞天真人说,"大会马上就要开始了。"

四人走进城中,秦寒跟随在洞天真人身后,白羽和慕容雪走在两侧。一路上,无数修士投来目光,有羡慕,有嫉妒,也有敌意。

"那是洞天真人!"
"三十六洞天的主人!"
"他身边那些年轻人是谁?"

秦寒能够感受到周围投来的目光,但他保持着平静。他知道,从今天开始,自己将站在风口浪尖上。

# 第一百三十七章 购买资源

时间:万宝大会第一天
地点:万宝广场交易区
人物:秦寒、摊主

洞天真人让三人自由行动,自己去见几个老朋友。

秦寒独自在广场上逛着。他对法宝不感兴趣,主要想找元婴丹和阵法相关的资源。

逛了一会儿,秦寒在一个摊位前停下了脚步。摊主是个老者,看起来平平无奇,但摊位上摆放的东西却很不错。

"小友,看中什么了?"老者笑眯眯地问。

秦寒的目光落在摊位上的一枚丹药上。那枚丹药通体晶莹,散发着浓郁的灵气。

"这是四阶元婴丹?"秦寒惊讶道。

四阶丹药,那可是能够给元婴初期强者使用的!而且元婴丹是最难炼制的丹药之一,成功率极低。

"小友好眼力。"老者笑道,"这正是四阶元婴丹。不过价格不菲,需要一万上品灵石。"

一万上品灵石!这相当于一百万中品灵石,对于金丹期修士来说是一笔巨款。

秦寒摸了摸储物戒,里面有三万上品灵石,是洞天真人给他的。这次大会,师父给了他五万上品灵石,让他购买需要的东西。

"我买了。"秦寒毫不犹豫地说。

"痛快!"老者大笑。

就在秦寒准备交易时,一个冰冷的声音响起:"这枚元婴丹,我要了。"

秦寒转头,看到一个身穿黑袍的年轻人走了过来。修为达到了金丹后期巅峰,而且气息阴冷,显然是修炼魔功的。

"这位道友,是我先看中的。"秦寒淡淡道。

"那又如何?"黑袍人冷笑,"在中界,拳头大就是道理。我要它,它就是我的。"

"是吗?"秦寒眼中闪过一丝寒芒。

# 第一百三十八章 大会激战

时间:万宝大会第一天
地点:万宝广场
人物:秦寒、黑袍人、围观修士

"既然你想抢,那就来试试。"秦寒冷冷道。

黑袍人哈哈大笑:"就凭你一个金丹后期?好,我成全你!"

话音未落,他突然出手了!

一只漆黑的手掌从天而降,带着恐怖的魔气,向秦寒抓来。周围的空间都因为这只手掌的力量而扭曲,显然这是极其强大的一招。

"魔道功法?"秦寒眉头微皱。

他没有退避,而是直接出拳。金色的灵力从他体内爆发出来,与黑袍人的攻击正面碰撞。

砰!

巨大的声响传遍整个广场,无数修士被这股力量震退。摊主老者连忙撑起护罩,保护自己的摊位。

烟尘散去,秦寒和黑袍人都后退了数步。

"有点意思。"黑袍人眼中闪过一丝惊讶,"没想到你居然能挡住我的攻击。"

"该我了。"秦寒说。

他打开命轮眼,双眼之中,金色的纹路浮现。在命轮眼的作用下,他的实力再次提升。

"这是...命轮眼?"黑袍人脸色大变,"你是那个秦寒?"

周围围观的修士们也震惊了。

"命轮眼?难道就是三十六洞天试炼冠军的那个秦寒?"
"听说他刚刚突破到金丹后期,就能和金丹后期巅峰一战!"
"居然真的有命轮眼!"

秦寒没有理会周围的议论,他凝聚灵力,一拳轰出。

这一拳蕴含着命轮眼和《苦海经》的力量,威力远超之前。黑袍人来不及闪避,只能硬接。

砰!

他直接被轰飞出去,重重摔在地上,口吐鲜血。

"怎么可能..."黑袍人不可置信地看着秦寒,"我才金丹后期巅峰,你怎么可能打败我?"

"因为我不只是金丹后期。"秦寒淡淡道。

在命轮眼的加持下,他的实力已经可以媲美金丹后期巅峰!

黑袍人挣扎着站起来,眼中闪过恐惧:"我认输,元婴丹给你了。"

说完,他头也不回地跑了。

周围围观的修士们爆发出一阵议论。秦寒的实力,让所有人都感到震撼。

# 第一百三十九章 万宝大会的收获

时间:万宝大会后
地点:洞天仙府
人物:秦寒、洞天真人

万宝大会持续了整整七天,秦寒终于回到了洞天仙府。

回想起这七天的经历,秦寒仍然心潮澎湃。

**第一天:大会开幕**

万宝城位于中界中部,是中界最繁华的城市。城池占地极广,足有下界十个玄天宗那么大。城墙高达千丈,通体由星辰铁打造,上面刻画着无数防御阵法。

大会开幕那天,秦寒跟随洞天真人来到万宝城。城中人山人海,无数修士从四面八方赶来。有元婴期的老怪,有金丹期的天骄,甚至还有化神期的绝世强者。

"这么多修士..."秦寒震撼了。下界最大的宗门也不过几千人,而这里单单参加大会的修士就有数万之众。

"这还只是一小部分。"慕容雪说,"万宝大会十年一次,整个中界的修士都会来。最多的时候,参会人数超过十万。"

**第三天:万宝楼淘宝**

第三天,秦寒来到了万宝楼。这是万宝城最大的商铺,也是整个中界最大的交易场所。楼高九十九层,每一层都卖不同阶的宝物。

秦寒直奔四楼,那里出售四阶丹药。他需要四阶元婴丹,这是突破元婴期的关键丹药。

"四阶元婴丹,一颗十万上品灵石。"柜台后的伙计淡淡道。

"十万!"秦寒心中一震。这价格太贵了,几乎是他全部身家。

"公子,四阶元婴丹可不是普通丹药。"伙计说,"它能将突破元婴期的成功率提高三成。整个中界,能炼制此丹的炼丹大师不超过十人。物以稀为贵,这个价格已经很公道了。"

秦寒咬咬牙,决定买下。虽然价格昂贵,但元婴期的突破至关重要,不能省这笔钱。

他取出储物戒,将十万上品灵石交给伙计。伙计验过灵石,这才小心翼翼地取出一枚晶莹剔透的丹药。丹药散发着淡淡的灵光,光是闻一闻都能感觉到灵气在涌动。

"这是上品元婴丹,品质极佳。"伙计说,"公子运气不错,我们这里只剩三颗了。"

买完元婴丹,秦寒又逛了其他楼层。在五楼,他看中了三件阵法法宝。

《千机百变阵》售价三万上品灵石,《五行困杀阵》也是三万,《空间封禁阵》两万。这三件阵法加起来就要八万上品灵石,价格不菲。

"三件一起买,能优惠吗?"秦寒问。

"公子爽快。"伙计笑道,"三件一起买,收您五万上品灵石。"

秦寒点头成交。虽然花费巨大,但阵法在关键时刻能救命,这笔投资绝对值得。

**第五天:遭遇挑衅**

大会第五天,秦寒在万宝楼外遇到了麻烦。

一名黑袍修士拦住了他的去路:"这位师弟,我看你买了不少好东西,不如让师兄我帮你保管?"

此人修为达到金丹后期巅峰,比秦寒高出不少。而且他还有三个同伙,都是金丹后期修为。

"让路。"秦寒冷冷道。

"哟,还挺横?"黑袍人冷笑,"小子,我劝你识相点。把储物戒交出来,我们还能放过你。否则..."

"否则如何?"秦寒握紧拳头。

"否则别怪我们不客气!"黑袍人猛地出手,一道黑色掌印向秦寒轰来。

秦寒眼神一凝,开启命轮眼。瞬间,他看到了黑袍人的攻击轨迹,向左一侧,轻松躲过。

"什么?"黑袍人震惊。

就在他惊讶之际,秦寒已经反击。苦海命运剑出现在手中,一剑刺出,剑光如虹。

黑袍人连忙防御,但秦寒的剑太快太猛,直接破开他的防御,在他胸口留下一道血痕。

"怎么可能!"黑袍人后退几步,难以置信。

"一起上!"他的三个同伙也出手了。

四人围攻秦寒,气势惊人。但秦寒不慌不忙,施展《步天诀》,身影在四人之间闪动,如同鬼魅一般。

四人根本打不中秦寒,反而被秦寒抓住机会,每人一剑,全部受伤。

"撤!"黑袍人意识到不是对手,带着同伙逃走。

周围围观的修士纷纷叫好,有人还鼓起了掌。

"好身法!"
"好剑法!"
"这位师弟叫什么名字?"

秦寒没有理会众人的议论,收起苦海命运剑,转身离去。但他不知道的是,这场战斗已经被很多人记住,他的名声也开始在万宝城传播。

**第七天:大会闭幕**

第七天,万宝大会闭幕。秦寒收获满满,不仅买到了元婴丹和阵法法宝,还增长了不少见识。

不过,他也认识到了自己的不足。在大会上,他遇到了太多比他强的修士。金丹后期巅峰比比皆是,元婴初期也不少见。甚至还有几个元婴中期的绝世天骄。

"还需要继续努力啊。"秦寒感慨。

这次大会,他收获颇丰。

除了元婴丹和阵法法宝,秦寒还遇到了不少有趣的事情。

比如他遇到一位自称来自东海的散修,对方邀请他一起去探索东海深处的秘境。秦寒婉言拒绝了,毕竟他现在的实力还不够,探索秘境太危险。

又比如他遇到了一位炼丹大师,对方看中了秦寒的灵力体质,想要收他为徒。秦寒也婉言拒绝了,因为他已经有师父了,不能背叛师门。

总体来说,这次万宝大会让秦寒大开眼界。他见识到了中界的繁华,也认识到了自己的不足。虽然他现在是金丹后期,但在大会上只能算是中等水平。很多金丹后期巅峰、甚至元婴初期的修士比比皆是。

"还需要继续努力啊。"秦寒感慨道。

不过,他也不气馁。毕竟他才修炼几年,能达到现在的成就已经是奇迹了。只要继续努力,将来一定能站在中界的巅峰。

但他没有立即突破,而是决定先回下界一趟。

自从来到中界,他已经快半年没有回去了。虽然可以通过传音玉简联系,但终究不如见面来得真切。父亲的身体如何,苏清歌的废体有没有改善,赵灵儿的修为进展如何...这些都让秦寒牵挂。

秦寒来到洞天殿,向洞真人辞行。

"师父,弟子想先回下界看看父亲和苏清歌她们。"秦寒恭敬道,"万宝大会后离家已久,想念她们了。而且我也想在下界突破元婴期,那里更加熟悉,心也更静。"

洞天真人点点头:"也好,你确实该回下界看看了。不过要小心,虽然你现在是金丹后期,在下界已经算是一方强者,但也不能大意。下界卧虎藏龙,老一辈的高手不少。"

"弟子明白。"秦寒说。

"还有。"洞天真人沉吟片刻,"你突破元婴期后,可以考虑返回中界。为师这里有一些元婴期的任务,正好可以让你历练。等你达到元婴后期,老夫可以推荐你去参加'天骄榜'争夺战。"

"天骄榜?"秦寒疑惑。

"那是中界年轻一代的排名榜单。"洞天真人解释道,"每年更新一次,收录元婴期以下的天才修士。如果能上榜,不仅能获得丰厚奖励,还能在整个中界扬名。"

秦寒心中一动。天骄榜,如果能上榜,就能在整个中界扬名。到时候寻找父亲遗骸的事情也会更容易。

"弟子一定努力!"秦寒郑重道。

洞真人欣慰地点点头:"去吧,记住,无论何时都不要骄傲自满。修真之路漫漫,一步走错就可能万劫不复。"

"弟子谨记师父教诲!"秦寒深深一拜。

他走出洞天殿,看到慕容雪和白羽都在外面等候。

"秦师弟,要回下界了吗?"慕容雪问道。

"是的。"秦寒点头,"有些事需要处理。"

"那就祝师弟一路顺风。"慕容雪难得露出一丝微笑。

白羽则是冷哼一声:"在下界可别丢了我们洞天仙府的脸。"

秦寒淡然一笑:"白师兄放心,我不会给师父丢脸的。"

三人简单告别后,秦寒踏上了返回下界的路程。

# 第一百四十章 返回下界

时间:离开中界后
地点:下界
人物:秦寒、宗主、父亲

秦寒来到洞天山外的空间传送阵,这是连接中界和下界的唯一通道。

传送阵占地约百丈,由无数星辰铁打造,上面刻画着繁复的阵纹。想要启动传送阵,需要消耗大量的灵石。以秦寒现在的修为,单程就需要消耗十块上品灵石。

"启动!"秦寒将灵石放入传送阵的凹槽。

轰隆隆——

传送阵发出低沉的轰鸣,一道光芒冲天而起,将秦寒笼罩其中。空间开始扭曲,他的身影逐渐模糊。

下一刻,秦寒出现在下界的传送阵中。

刚一下界,他就感受到了熟悉的灵气。下界的灵气虽然不如中界浓郁,但却更加纯净,更加适合修炼。空气中弥漫着泥土和草木的芬芳,这是中界所没有的。

"终于回来了。"秦寒深吸一口气,感慨道。

他驾起遁光,向玄天宗的方向飞去。

下界的山川河流在脚下掠过,一切都那么熟悉。秦寒离开虽然只有半年,但感觉像过了很久。中界的生活虽然精彩,但下界才是他的根。

不到一个时辰,秦寒就回到了玄天宗。

玄天宗依然如旧,山门巍峨,云雾缭绕。外门弟子们正在操场上修炼,看到秦寒驾着遁光从天而降,所有人都震惊了。

"那是...秦寒长老?"
"他回来了!"
"真的是秦寒长老!"

消息很快传遍了整个宗门。内门弟子、核心弟子纷纷出来观看,都想一睹这位玄天宗最年轻长老的风采。

宗主听说后,亲自从主殿出来迎接。

"秦寒,你回来了!"宗主激动地上前,"在中界还好吗?"

"很好,多谢宗主关心。"秦寒拱手行礼,"这些年多亏宗主照顾我父亲,秦寒感激不尽。"

"应该的,应该的。"宗主笑道,"你父亲是我宗门的贵客,照顾他是应该的。对了,你父亲现在在后山的洞府修炼,我带你去见他。"

"有劳宗主。"秦寒说。

两人向后山飞去。一路上,宗主向秦寒介绍宗门的情况。原来在秦寒离开的这段时间,玄天宗发展得很不错。不仅招收了不少新弟子,还有几名弟子突破到了金丹期,宗门实力大增。

"这些都是托你的福。"宗主笑道,"自从你在中界成名后,很多年轻修士都想加入玄天宗。他们说玄天宗出了个天才,将来一定能成为一方霸主。"

秦寒苦笑:"宗主过誉了,我只是运气好而已。"

说话间,两人已经来到后山的洞府前。这里是玄天宗灵气最浓郁的地方,专门供父亲居住。

洞府门开着,秦寒看到父亲正在修炼。父亲的头发已经花白,但精神依然矍铄。看到秦寒和宗主进来,他睁开眼睛。

当看到秦寒的那一刻,父亲激动得站了起来,身体微微颤抖。

"寒儿..."父亲声音哽咽。

"父亲!"秦寒再也控制不住,冲过去一把抱住父亲。

父子二人相拥而泣,场面感人至深。宗主见状,悄悄退了出去,把空间留给父子二人。

良久,父子二人才平复情绪。

"寒儿,你达到金丹后期了?"父亲震惊道。他能感受到秦寒身上散发的气息,比半年前强大了数倍。

"是的。"秦寒点头,"在中界这几个月,弟子修为有了突破。"

"好,好!"父亲激动得热泪盈眶,"你母亲在天之灵,看到你今天的样子,一定会很欣慰。"

秦寒心中一酸。母亲虽然早逝,但她的期望一直是他前进的动力。每次修炼遇到困难时,想起母亲临终前的话语,他就能重新振作。

"对了,父亲。"秦寒突然想起什么,"清歌和灵儿呢?她们还好吗?"

提到苏清歌和赵灵儿,父亲脸上露出笑容:"她们啊,都很好。清歌这孩子很刻苦,虽然没有天生废体,但她一直在努力修炼。现在已经突破到凝气期九重了。"

"灵儿就更不用说了。"父亲笑道,"她化形后修炼速度极快,现在已经筑基初期了。不过她性子活泼,整天在宗门里到处跑,倒是给清歌添了不少麻烦。"

刚到院门口,秦寒就听到里面传来说话声。

"灵儿,你慢点,别摔着了。"一个温柔的女声响起。那是苏清歌的声音。

"嘻嘻,清歌姐,秦寒哥哥就要回来了,我高兴嘛!"另一个清脆的声音说道。那是赵灵儿。

秦寒心中一暖,推开院门。

院子里,一名白衣少女正在晾晒药材。她约莫十八九岁,容貌清秀,气质温婉。虽然修为只有凝气期九重,但身上散发着一种宁静的气息,让人心神安宁。

旁边站着一名少女,约莫十六七岁,容貌精致可爱,一双大眼睛灵动有神。她穿着一身淡青色的长裙,腰间系着一条红色的丝带,显得活泼又俏皮。

"清歌,灵儿。"秦寒轻声唤道。

两女闻声回头,当看到秦寒时,都愣住了。

"秦寒哥哥!"赵灵儿第一个反应过来,欢快地冲了过来,一把抱住秦寒的手臂,"你终于回来了!我想死你了!"

秦寒笑着揉了揉她的头:"灵儿又长高了。"

"那当然!"赵灵儿骄傲地挺起胸,"我现在可是筑基初期了!"

苏清歌也走了过来,眼中闪烁着泪光:"秦寒...你回来了。"

"嗯,我回来了。"秦寒看着她,心中涌起一股暖意。

苏清歌虽然天生废体,修炼极其困难,但她从不放弃。这半年来,她一定过得很辛苦吧。

"清歌,你的身体..."秦寒关切地问。

"还好。"苏清歌淡淡一笑,"虽然不能像其他人那样修炼,但我也在努力。现在已经凝气期九重了,距离筑基期也不远了。"

"清歌姐太谦虚了!"赵灵儿抢着说,"她可是花了整整半年才从凝气八重突破到九重,真的很不容易!"

秦寒心中一酸。普通人从凝气八重到九重,最多一个月。而苏清歌花了半年,可见她的修炼有多困难。

"清歌,我有办法帮你。"秦寒说,"我在中界学到了一些特殊的法门,应该能改善你的体质。"

"真的?"苏清歌眼睛一亮。

"真的。"秦寒点头,"不过需要等时间。我这次回来,主要是准备突破元婴期。等我突破后,就会帮你改善体质。"

"太好了!"苏清歌激动得流下眼泪。

"清歌姐,你哭什么呀!"赵灵儿连忙给她擦眼泪,"这是好事,应该高兴才对!"

"我...我这是高兴的。"苏清歌破涕为笑。

三人相视而笑,温馨的氛围在院子里弥漫。

在苏清歌的小院待了半个时辰,秦寒告别两女,回到父亲的洞府。

看到秦寒回来,父亲问道:"见到清歌和灵儿了?"

"见到了。"秦寒点头,"她们都很好。清歌虽然修炼困难,但一直在努力。灵儿化形后很活泼,给宗门增添了不少生气。"

"那就好。"父亲欣慰地笑了,"清歌这孩子不容易,天生废体,修炼极其困难。但她从不放弃,这份毅力让人佩服。"

"父亲,这次回来,我准备突破元婴期。"秦寒说,"突破之后,我就能更好地保护您和大家了。"

"元婴期..."父亲感慨万千,"当年的我,也曾梦想突破元婴期。可惜..."

说到这里,父亲的眼神黯淡下来。

地点:玄天宗后山洞府
