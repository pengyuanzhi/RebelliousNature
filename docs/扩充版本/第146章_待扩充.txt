# 第一百四十六章 父亲痊愈

时间:疗伤后数日
地点:玄天宗,长老洞府
人物:秦寒、父亲

经过几天的调养,父亲的旧伤基本痊愈了。

这一天,秦寒再次检查父亲的身体,发现心脉和丹田的旧伤已经完全愈合,体内灵力运行也恢复了正常。

"父亲,您的伤完全好了!"秦寒高兴地说。

父亲感受了一下体内的灵力,确实顺畅了许多。以前的种种不适——胸闷、气短、灵力凝滞——全都消失了。

"太好了!"父亲激动地说,"寒儿,真的谢谢你!"

"父亲,您是我父亲,不用谢。"秦寒笑道。

"不,这次真的要好好谢谢你。"父亲认真地说,"这些旧伤跟了我几十年,让我吃尽了苦头。如果不是你,我恐怕这辈子都好不了了。"

"父亲,您为我付出了一切,我为您治疗一点伤,又算得了什么?"秦寒说。

父亲眼中闪过泪光。

他想起当年秦寒出生时,家中一贫如洗。为了给秦寒买米,他每天天不亮就上山砍柴,然后挑到镇上去卖,风里来雨里去,从不抱怨。

后来母亲去世,他更是承担起所有责任,既要照顾秦寒,又要维持生计。那些年,他真的很累,但从未放弃。

"寒儿,你长大了。"父亲感慨地说,"不仅修炼到了元婴期,还这么孝顺。你母亲如果在天有灵,一定会很欣慰。"

"父亲,您别这么说。"秦寒说,"如果不是您,我不可能有今天。您是我生命中最重要的人。"

"好,好。"父亲连连点头,"对了,寒儿,你的修为已经巩固得怎么样了?"

"差不多了。"秦寒说,"我准备继续闭关,争取突破到元婴中期。"

"好,你以修炼为重,不用管我。"父亲说,"我现在身体好了,可以照顾自己。"

"父亲,我会常来看您的。"秦寒说。

"嗯,去吧。"父亲点头。

秦寒回到修炼室,准备闭关。

不过在闭关之前,他还有一件事要做——命轮眼。

这段时间以来,命轮眼的能力越来越强。秦寒发现,他不仅能看破虚妄,还能预知危险,甚至能看到一些模糊的未来片段。

"命轮眼..."秦寒喃喃自语。

他知道,命轮眼是他的金手指,是他最大的依仗。但他也深知,命轮眼的真正威力,他还远远没有发挥出来。

"等我的修为再高一些,一定要彻底开发命轮眼的力量。"秦寒暗暗下定决心。

就在这时,他突然心有所感,看向窗外。

远处,天边出现了一道流光。

"那是..."秦寒眯起眼睛。

流光越来越近,秦寒终于看清了,那是一个人!

那人速度极快,眨眼间就到了玄天宗上空。

"何人擅闯我宗!"秦寒厉喝。

那人停下,看向秦寒。

秦寒也看清了那人的样貌——是一位老者,须发皆白,身穿道袍,气息深不可测。

"你是..."秦寒警惕地问。

"哈哈哈哈!"老者大笑,"小娃娃,你是洞天真人的弟子吧?"

秦寒心中一震:"你怎么知道?"

"老夫就是洞天真人的好友,特地来找你。"老者说。

"找我?有什么事吗?"秦寒问。

"洞天真人让老夫给你带个话。"老者说,"中界将有大变,让你速速返回!"

# 第一百四十七章 父子深谈

时间:神秘老者到来后
地点:玄天宗,长老洞府
人物:秦寒、父亲、神秘老者

秦寒邀请老者进入洞府,奉上灵茶。

"前辈,请问中界将发生什么大事?"秦寒问。

老者喝了一口茶,缓缓道:"小娃娃,你知道中界之外,还有上界吗?"

"上界?"秦寒心中一动。

父亲曾经告诉他,当年去上界寻找母亲,受了重伤才返回。难道上界真的存在?

"是的,上界。"老者说,"中界不过是下界的飞升之地,真正的修真大世界,在上界。每隔百年,上界就会开启一次飞升通道,让中界的顶尖修士飞升。"

"那这次的大变,是指飞升通道开启?"秦寒问。

"不完全是。"老者摇头,"这次的情况比较特殊。据老夫所知,上界内部将发生重大变故,可能会影响到中界甚至下界。洞天真人担心你受到影响,所以让老夫来提醒你。"

"多谢前辈提醒。"秦寒拱手,"请问前辈,我需要做什么准备吗?"

"你现在的修为还太低。"老者直言不讳,"元婴初期,在下界或许算强者,但在中界,不过是中下水平。如果上界真的发生变故,你根本自保不了。"

秦寒沉默了。

老者说得对,他现在确实太弱了。

"所以,老夫建议你,"老者继续说,"尽快提升修为。至少要达到元婴后期,才有资格参与这次的大事件。"

"元婴后期..."秦寒喃喃自语。

"是的。"老者说,"小娃娃,老夫知道这对你来说很难,但这是现实。修真界弱肉强食,没有实力,只能被人踩在脚下。"

秦寒握紧拳头。

他明白这个道理。从小到大,他一直在为变强而奋斗。母亲去世、父亲受伤、婆婆牺牲——这些苦难,都让他明白一个道理:弱小,就是原罪!

"前辈放心,我会尽快提升修为的。"秦寒坚定地说。

"好,有志气!"老者赞许地点头,"老夫还有一事相求。"

"前辈请说。"

"你父亲,当年去上界,是为了寻找你母亲吧?"老者问。

秦寒和父亲都是一震。

"你怎么知道?"父亲震惊地问。

"老夫当年也在上界。"老者说,"当年看到过一个下界修士,狼狈逃回,受了重伤。后来一打听,才知道是为了寻找亡妻。老夫很感动,所以印象深刻。"

"是的。"父亲点头,"我当年确实去上界寻找亡妻,但实力太弱,不仅没找到,还受了重伤。"

"你妻子,叫什么名字?"老者问。

"秦林氏,名唤林婉儿。"父亲说。

老者沉思片刻,突然脸色一变:"你是说,林婉儿是你妻子?"

"是的。"父亲点头,"前辈认识她?"

老者深吸一口气:"这么说来,你儿子就是...那个命轮眼拥有者?"

秦寒心中一震:"前辈知道命轮眼?"

"何止知道。"老者感叹道,"小娃娃,你知道命轮眼的来历吗?"

秦寒摇头。

"命轮眼,是上古时期最为神秘的神通之一。"老者说,"拥有命轮眼的人,注定不凡。你母亲当年,就是为了保护命轮眼的秘密,才被妖兽杀死的。"

"什么?!"秦寒如遭雷击。

母亲是为了保护命轮眼的秘密,才被妖兽杀死的?这怎么可能?

老者看着秦寒震惊的表情,叹了口气:"看来你还不知道。小娃娃,你母亲不是普通的凡人,她是命轮一族的遗脉!"

# 第一百四十八章 闭关修炼

时间:得知身世后
地点:玄天宗,长老洞府修炼室
人物:秦寒

秦寒回到修炼室,心绪难平。

母亲是命轮一族的遗脉?母亲是为了保护命轮眼的秘密,才被妖兽杀死的?

这些信息,对秦寒来说,如同晴天霹雳。

从小到大,他一直以为母亲只是普通的凡人,被妖兽杀死是意外。但现在看来,母亲的死,并非意外,而是有人为了得到命轮眼的秘密,故意为之!

"可恶!"秦寒握紧拳头,眼中闪过怒火。

"母亲,您到底还有多少事情瞒着我?"

秦寒闭上眼睛,深吸一口气,努力让自己冷静下来。

愤怒不能解决问题,只有实力,才能查清真相!

"不管是谁害死了母亲,我一定会为他报仇!"

秦寒盘膝坐下,开始修炼。

体内的元婴缓缓旋转,吸收着周围的灵气。元婴期的修炼,与金丹期有很大不同。金丹期主要靠金丹炼化灵气,而元婴期,则是靠元婴直接吸收和炼化。

秦寒运转《苦海经》,这门功法越来越深奥。他发现自己对《苦海经》的理解还远远不够,这部功法蕴含的大道至理,需要慢慢领悟。

"以苦炼心,以痛证道..."秦寒喃喃自语。

《苦海经》的核心,就是将所有的痛苦转化为力量。秦寒回想起自己经历的所有苦难——母亲去世、父亲受伤、婆婆牺牲、修炼艰辛...

这些痛苦,刻在他的心上,成为了永恒的伤疤。

但现在,这些伤疤不再是伤痛,而是力量!

每一次回想,秦寒都能感受到一股力量从心底涌出,这股力量让他的修为不断提升。

轰!

秦寒的修为开始突破。

元婴初期巅峰...元婴中期初期...元婴中期初期巅峰...

短短一个月时间,秦寒就从元婴初期初期,突破到了元婴中期初期!

"太快了!"秦寒惊喜道。

正常情况下,从元婴初期突破到元婴中期,需要数年甚至数十年时间。但秦寒只用了一个月!

《苦海经》的威力,真是恐怖!

不过,秦寒也清楚,这种快速突破,是因为他之前积累了足够多的底蕴。金丹期到元婴期,他花了数年时间巩固,根基非常扎实。所以现在突破起来,才会如此顺利。

"继续!"秦寒没有停下。

他知道,时间紧迫。中界将发生大事,上界也可能有变故。他必须在最短的时间内,提升到元婴后期,甚至元婴圆满!

只有这样,他才有资格参与到即将到来的大事件中!

命轮眼的力量,也要彻底开发!

秦寒继续修炼,不知疲倦。

一天又一天,一月又一月...

三个月后,秦寒终于再次突破!

元婴中期!

他成功从元婴中期初期,突破到了元婴中期中期!

"又突破了一小境界!"秦寒握紧拳头。

现在的他,实力比三个月前强大了数倍。神识范围扩大到方圆百里,一招一式都蕴含着毁天灭地的威力。

"还不够!"秦寒摇头,"必须继续!"

他再次进入修炼状态,继续冲击元婴中期后期。

# 第一百四十九章 巩固修为

时间:闭关三个月后
地点:玄天宗,长老洞府修炼室
人物:秦寒、父亲

秦寒结束闭关,走出修炼室。

三个月的闭关,让他的修为从元婴初期初期,提升到了元婴中期中期。这样的速度,简直骇人听闻!

不过,连续突破后,秦寒也感到有些疲惫。他决定先稳固修为,然后再继续突破。

走出洞府,秦寒看到父亲正在院子里浇花。

"父亲!"秦寒喊道。

父亲抬头,看到秦寒,脸上露出笑容:"寒儿,你出关了?"

"是的,父亲。"秦寒点头,"三个月的闭关,我的修为有所突破。"

"哦?突破到什么境界了?"父亲好奇地问。

"元婴中期中期。"秦寒说。

"什么?!"父亲震惊得手中的水壶都掉了,"元婴中期中期?!"

"是的。"秦寒点头。

父亲良久才回过神来:"寒儿,你的修炼速度...也太快了吧?从元婴初期到元婴中期中期,你只用了三个月?"

"是的。"秦寒说,"不过,这是因为我在金丹期和元婴初期,都积累了足够的底蕴。所以现在突破起来,才会这么快。"

"天呐..."父亲感叹道,"你母亲如果在天有灵,一定为你骄傲!"

"父亲,我会继续努力的。"秦寒说,"我的目标是元婴后期,甚至元婴圆满!"

"好,有志气!"父亲赞许地点头,"不过,修炼也不能太急。你现在刚突破,需要好好巩固。否则根基不稳,将来难以寸进。"

"父亲说得是。"秦寒点头,"所以我打算先稳固修为,然后再继续突破。"

"嗯,这样就好。"父亲说,"对了,你这次出关,准备待多久?"

"大概一个月吧。"秦寒说,"一个月后,我准备返回中界。"

"这么快?"父亲有些不舍。

"父亲,中界将发生大事,我必须回去。"秦寒说,"不过您放心,等我在中界稳定下来,一定来接您过去。"

"好,我等你。"父亲点头。

父子二人聊了很久。父亲告诉秦寒,他这三个月在宗门过得很好,宗主和长老们都很照顾他。而且父亲的旧伤痊愈后,身体状况越来越好,甚至可以修炼了。

"父亲,您也可以修炼?"秦寒惊喜道。

"是的。"父亲笑道,"虽然我的资质一般,但慢慢修炼,总能有所进步。说不定哪天,我也能突破到凝气期呢。"

"太好了!"秦寒高兴地说,"父亲,如果您需要修炼资源,尽管跟我说。我现在是长老,可以进入宗门秘境,那里有很多宝物。"

"好,好。"父亲点头。

接下来的一个月,秦寒没有继续突破,而是专心巩固修为。

每天除了陪伴父亲,就是在宗门内走动,熟悉长老的身份和职责。宗主和长老们对秦寒都很客气,毕竟他不仅是元婴期修士,还是洞天真人的弟子,将来前途无量。

一个月后,秦寒的修为已经彻底稳固。

"父亲,我要走了。"秦寒说。

"去吧,以修炼为重。"父亲点头,"不用担心我,我会照顾自己的。"

"父亲,您保重。"秦寒抱了抱父亲。

"你也是,照顾好自己。"父亲说。

秦寒告别父亲和宗主,准备返回中界。

在离开之前,他还有一件事要做——去太华宗,和苏清歌告别。

# 第一百五十章 第二卷完

时间:离开下界前
地点:太华宗,青州城
人物:秦寒、苏清歌

秦寒来到太华宗,再次见到了苏清歌。

"秦寒哥哥!"苏清歌看到他,眼中闪过惊喜。

"清歌,我来向你告别。"秦寒说。

"告别?你要去哪里?"苏清歌问。

"我要返回中界了。"秦寒说,"中界将发生大事,我必须回去。"

"哦..."苏清歌有些失落,"那你什么时候回来?"

"不确定。"秦寒说,"不过你放心,等我在中界稳定下来,一定来接你过去。"

"嗯,我等你。"苏清歌点头。

两人沉默了一会儿。

"清歌,你要继续努力修炼。"秦寒说,"争取早日突破到元婴期,这样我接你过去时,你也有自保之力。"

"好,我会的。"苏清歌说,"秦寒哥哥,你也要小心。中界肯定比下界危险,你一定要保护好自己。"

"放心吧。"秦寒笑道,"我现在可是元婴期修士,一般的危险,我不怕。"

"嗯,但还是小心为好。"苏清歌还是不放心。

秦寒看着苏清歌,心中涌起一股暖意。

从认识她到现在,已经过去好几年了。当年的青州城相遇,仿佛还在昨天。那时的苏清歌,还是个柔弱的少女,需要他的保护。而现在的苏清歌,已经成长为金丹后期修士,能够独当一面了。

"清歌,等我。"秦寒认真地说。

"嗯,我等你。"苏清歌也认真地说。

两人相拥而别。

秦寒离开太华宗,飞向中界传送阵。

传送阵位于青州域最东端,是一座古老的阵法。据说,这是上古时期的大能留下来的,专门用来连接下界和中界。

秦寒来到传送阵前,深吸一口气。

"下界,我还会回来的。"秦寒在心里默默说道,"等我成为真正的强者,一定回来,保护我想保护的一切!"

他踏上传送阵,灵力注入其中。

轰!

传送阵启动,一道白光笼罩秦寒。下一刻,他的身影消失在原地,出现在另一个世界。

中界,他又回来了!
