# 第一百章 小九化形

时间:参加拍卖会归来
地点:玄天宗洞府
人物:秦寒、苏清歌、小九

拍卖会之行,秦寒和苏清歌收获颇丰。

虽然没能找到七窍玲珑心,但找到了混沌原石和万年灵草。

还买到了一些珍贵的法宝和功法。

"至少不是空手而归。"秦寒安慰自己。

"是啊。"苏清歌笑道,"而且,我们还赚了不少钱。"

原来,他们在拍卖会上,卖掉了一些不需要的东西,换了不少灵石。

回到玄天宗后,两人继续修炼。

就在这时,小九突然发出了痛苦的声音。

"小九,你怎么了?"秦寒关切地问道。

小九倒在地上,身体开始发生变化。

它的身形在缩小,狐毛在脱落,爪子在变成手脚...

"它在化形!"苏清歌惊喜道。

秦寒连忙布置阵法,保护小九。

化形是一个危险的过程,如果被打扰,可能会失败。

半个时辰后,化形完成了。

小九变成了一个绝美的少女。

看起来约莫十六七岁,皮肤雪白,眼睛灵动,穿着秦寒为她准备的衣裙。

"小九...不对,应该叫你灵儿了。"秦寒笑着说道。

"主人。"灵儿开口了,声音清脆悦耳,"我化形了!"

"是啊。"秦寒欣慰地笑了,"你终于化形了。"

"主人,谢谢你。"灵儿认真地说道,"如果不是你,我不可能化形得这么快。"

"这是我应该做的。"秦寒说道,"从你还是一只幼狐时,你就跟着我。我们是伙伴,是家人。"

"嗯。"灵儿幸福地点头。

苏清歌也走过来,摸了摸灵儿的头:"真可爱。"

"师娘好。"灵儿乖巧地叫道。

"真乖。"苏清歌笑道。

三人(一狐)相视而笑,心中充满了幸福。

从今以后,秦寒身边又多了一位红颜知己。

虽然灵儿是小九化形的,但在秦寒心中,她已经是家人了。

"灵儿,以后你就叫赵灵儿。"秦寒说道,"赵是你师娘的姓氏,灵儿是灵狐的意思。"

"好,我都听主人的。"灵儿乖巧地点头。

"不用叫我主人了。"秦寒笑道,"叫秦寒,或者师兄。"

"那...师兄?"灵儿试探地叫道。

"嗯。"秦寒满意地点头。

灵儿也笑了,眼中闪过一丝幸福。

从今以后,她不再是灵兽,而是人了。

可以和秦寒、苏清歌一起,走在修真路上。

三人相拥,看着窗外的星空。

未来的路,还很长。

但他们相信,只要在一起,就能度过任何难关。

# 第一百零一章 玄天宗的日常

时间:小九化形后
地点:玄天宗
人物:秦寒、苏清歌、赵灵儿

清晨的阳光洒在玄天宗的山峰上,给这座古老的宗门披上了一层金色的薄纱。

秦寒缓缓睁开眼睛,吐出一口浊气。金丹后期的修为在体内流转,让他感到前所未有的力量。

"小九化形已经三天了..."秦寒喃喃自语,"时间过得真快。"

三天前,小九成功化形为人,取名赵灵儿。那是一场盛大的庆典,整个玄天宗都为之震动。九尾天狐的化形,在修真界可是百年难遇的奇观。

秦寒走出洞府,看到赵灵儿正在院子里练习走路。

没错,走路。

虽然化形了,但赵灵儿还是不太适应人类的双足行走。她跌跌撞撞地走了几步,就差点摔倒。

"灵儿,不用着急。"秦寒笑着走过去,"慢慢来,你会习惯的。"

赵灵儿脸红了:"师兄,我还是不太习惯用两条腿走路...当狐狸的时候多方便,想跑就跑,想跳就跳。"

"当人也有当人的好处。"秦寒说道,"比如,你可以用双手做很多事情,可以穿漂亮的衣服,可以..."他顿了顿,"可以更好地照顾我们。"

赵灵儿眼睛一亮:"真的?"

"真的。"秦寒点头,"而且,你现在是人了,就要遵守人类的规矩。比如,不能随便在人前化为本体,不能在大街上乱跑,不能..."

"啊,这么多规矩啊!"赵灵儿抱怨道。

"慢慢学吧。"秦寒笑道。

就在这时,苏清歌从屋里走了出来。她穿着一身淡青色的长裙,长发随意地挽起,清新淡雅。

"秦寒,灵儿,你们在聊什么?"苏清歌笑着问道。

"在教灵儿学习人类的生活方式。"秦寒说。

"师娘!"赵灵儿开心地叫道,"你看,我会走路了!"

赵灵儿小心翼翼地走了几步,虽然还有些摇摇晃晃,但已经稳定多了。

"真厉害!"苏清歌赞叹道,"才三天就能走这么稳了。"

"嘿嘿。"赵灵儿不好意思地笑了笑。

三人在院子里坐了下来,享受着清晨的阳光。

"对了,灵儿。"秦寒说道,"你现在虽然是人了,但还需要好好修炼。只有不断修炼,才能变得更强。"

"是,师兄。"赵灵儿乖巧地点头,"我会努力的!"

秦寒开始传授赵灵儿基础的修炼功法。

"修炼的第一步,是感应灵气。"秦寒解释道,"你闭上眼睛,感受周围的灵气。"

赵灵儿照做,闭上眼睛。

"灵气无处不在。"秦寒的声音温和地说道,"在空气中,在大地上,在每一棵草、每一朵花中。你只需要静下心来,就能感受到它们。"

赵灵儿闭上眼睛,努力感受着。

渐渐地,她感觉到了。空气中仿佛有无数微小的光点在飞舞,它们温暖、柔和,让人感到舒适。

"我...我感受到了!"赵灵儿惊喜地睁开眼睛。

"很好。"秦寒赞许地点头,"现在,试着引导这些灵气进入你的身体。"

赵灵儿再次闭上眼睛,按照秦寒的教导,引导周围的灵气进入体内。

灵气顺着她的毛孔,慢慢流入经脉。一开始很艰难,但渐渐地,灵气开始在她的经脉中流转。

虽然赵灵儿是灵兽化形,但她的天赋极高,修炼速度远超常人。短短一个月,她就达到了淬体期三重!

"师娘,你看!"赵灵儿兴奋地向苏清歌展示,"我达到淬体期了!"

"真厉害!"苏清歌赞叹道,"才一个月就达到淬体期三重了,这速度简直不可思议。"

"这还不算什么。"秦寒笑道,"以灵儿的天赋,很快就能追上我们。不过,淬体期只是基础,后面的路还很长。"

"我会努力的!"赵灵儿坚定地说。

除了教导赵灵儿,秦寒也在巩固自己的金丹后期修为。

金丹后期是金丹期的最后一个小境界,巩固好后就可以准备突破元婴期了。

秦寒盘膝坐在蒲团上,运转《苦海经》第三层。

《苦海经》第三层对应元婴期,核心是"以苦炼神,以痛证心"。

元婴期,是修真路上的一个重要分水岭。突破元婴期后,修士将拥有第二条性命——元婴。

元婴是神识的能量体,拥有无穷的潜力。只要元婴不灭,就算肉身被毁,也能夺舍重生。

"元婴期..."秦寒喃喃自语,"我离这个境界,已经不远了。"

他开始研究元婴期的突破方法。

根据古籍记载,突破元婴期需要三个条件:

第一,金丹后期圆满。修为必须达到金丹期的巅峰,灵力储量达到极限。

第二,神识强大。元婴是神识的能量体,如果神识不够强大,就无法凝聚元婴。

第三,机缘。有些修士即使满足了前两个条件,也无法突破元婴期,因为缺少机缘。

"我现在已经满足前两个条件。"秦寒分析道,"至于第三个...我相信,机缘会来的。"

苏清歌的修炼也在稳步前进。

她的废体虽然还没有完全修复,但在双修的帮助下,已经有了明显的好转。她的修为也从金丹初期,提升到了金丹中期。

"清歌,你的进步很快。"秦寒欣慰地说。

"都是因为你的帮助。"苏清歌温柔地笑道,"如果不是你双修,我可能连金丹初期都突破不了。"

"这是我们共同努力的结果。"秦寒握住她的手,"我相信,总有一天,你的废体会完全修复。到时候,你就能和我一起,站在修真界的巅峰。"

"嗯。"苏清歌幸福地点头。

日子就这样一天天过去。

秦寒巩固修为,苏清歌继续修炼,赵灵儿努力适应人类生活。

三人的生活充实而温馨。

每当夜幕降临,三人就会聚在一起,谈论修炼的心得,分享彼此的故事。

有时候,赵灵儿会讲她在灵兽山时的趣事,逗得秦寒和苏清歌哈哈大笑。

有时候,苏清歌会讲丹塔的秘闻,让秦寒和赵灵儿听得入迷。

有时候,秦寒会讲他从小到大的经历,让苏清歌和赵灵儿为之动容。

"师兄,你小时候真不容易。"赵灵儿感慨道。

"是啊。"苏清歌也眼中含泪,"你能走到今天,真的很不容易。"

"都过去了。"秦寒笑了笑,"正是因为那些苦难,才让我变得更强大。"

"苦难不是阻碍,而是阶梯。"他认真地说道,"只有经历过最深的苦难,才能登上最高的山峰。"

苏清歌和赵灵儿都被他的话感动了。

"夫君/师兄说得对。"她们异口同声地说。

三人相视而笑,夜风吹过,带来阵阵花香。

这就是他们的日常。

虽然修炼之路充满艰辛,但只要三个人在一起,就什么都不怕。

# 第一百零二章 宗门任务

时间:修炼一段时间后
地点:玄天宗任务殿
人物:秦寒、苏清歌、赵灵儿、任务殿执事

转眼间,距离小九化形已经过去了一个月。

这一个月来,赵灵儿的进步神速。她不仅完全适应了人类的生活,修为也突破到了凝气期三重。这样的速度,在玄天宗的历史上都是罕见的。

这天清晨,秦寒来到玄天宗的任务殿。

任务殿位于玄天宗的外门,是一座宏伟的三层建筑。第一层是发布任务的地方,第二层是兑换奖励的地方,第三层则是长老们商议重要事务的地方。

秦寒走进任务殿,里面已经有不少弟子在浏览任务。

墙上挂着各种各样的任务牌,每个任务牌上都写着任务的内容、难度、奖励等信息。

"炼制回气丹×10,奖励贡献点100。"
"击杀二阶魔核×5,奖励贡献点150。"
"护送商队前往青州城,奖励贡献点200。"
...

秦寒浏览着这些任务,寻找适合自己的。

"秦师弟!"一个熟悉的声音响起。

秦寒转头,看到任务殿的执事长老正朝他走来。

"执事长老。"秦寒行礼道。

"秦师弟,你现在已经是金丹后期了,这些外门任务对你来说太简单了。"执事长老笑道,"跟我来,我带你看内门任务。"

秦寒跟着执事长老来到了任务殿的第二层。

这里的任务明显比第一层难得多,奖励也更加丰厚。

"调查青州域东部的古遗迹,奖励贡献点5000,上品灵石×100。"
"击杀四阶后期的魔兽,奖励贡献点3000,中品灵石×500。"
"寻找失落的古籍《化神论》,奖励贡献点10000,神阶功法×1。"
...

"这些都是内门任务。"执事长老解释道,"只有金丹期以上的修士才能接取。不过,考虑到秦师弟你现在的实力,我推荐一个任务给你。"

说着,执事长老带着秦寒来到一面特殊的墙前。

这面墙上只有三个任务,但每一个都非常特殊。

"秦师弟,你看。"执事长老指着中间的任务牌,"这个任务是青州域北部出现的魔兽动乱。根据情报,那里出现了五阶初期的魔兽,实力堪比金丹初期。而且,魔兽的数量还在不断增加,已经摧毁了好几个村庄。"

"这个任务的难度不小,但奖励也很丰厚。完成之后,可以获得贡献点10000,上品灵石×200,还有一枚破境丹。"

秦寒眼睛一亮:"破境丹?"

"没错。"执事长老点头,"破境丹是辅助突破境界的丹药,对于金丹期突破元婴期有很大帮助。以秦师弟你现在的修为,正是需要这种丹药的时候。"

秦寒想了想,点头道:"好,我接这个任务。"

"不过,我要提醒你。"执事长老认真地说,"这个任务非常危险。五阶初期的魔兽已经很强了,而且听说还有五阶中期的魔兽王。如果不幸遇到,就算是金丹后期也会很危险。"

"多谢长老提醒。"秦寒说道,"我会小心的。"

执事长老拿出一枚青铜令牌,递给秦寒:"这是任务令牌。拿着它,就可以证明你是接取了这个任务。完成任务后,带着令牌和魔兽的魔核来找我。"

"三个月内完成即可。"

秦寒接过令牌,收好。

"多谢长老。"

"不客气。"执事长老笑道,"对了,秦师弟,你可以带其他弟子一起去。不过,贡献点需要你们自己分配。"

秦寒想了想,点头道:"我会的。"

告别执事长老后,秦寒离开了任务殿。

刚走出任务殿,他就看到苏清歌和赵灵儿在等他。

"夫君,接了什么任务?"苏清歌关心地问道。

"调查青州域北部的魔兽动乱。"秦寒说道,"据说那里出现了五阶初期的魔兽。"

"五阶初期?!"赵灵儿惊讶道,"那可是金丹初期的实力啊!"

"没错。"秦寒点头,"所以很危险。你们..."

"我也要去!"苏清歌坚定地说道。

"我也要去!"赵灵儿也跟着说。

秦寒有些犹豫:"可是..."

"夫君,我们已经是道侣了。"苏清歌握住他的手,"无论你去哪里,我都要跟着。而且,我现在也是金丹中期,可以帮上忙的。"

"而且,我是丹塔的弟子,炼丹术还算不错。"苏清歌笑道,"路上如果需要疗伤丹药,我都可以炼制。"

"还有我!"赵灵儿跳了起来,"师兄,我现在虽然修为还不高,但我是九尾天狐啊!在感知方面,我可比你们强多了。我可以提前发现敌人的踪迹。"

秦寒看着两人坚定的眼神,心中一暖。

"好。"他点头,"那我们一起去。"

"耶!"赵灵儿欢呼起来。

"不过,要答应我一个条件。"秦寒认真地说,"遇到危险的时候,你们一定要听我的指挥。不要逞强,安全第一。"

"是!"两人齐声应道。

三人回到洞府,开始准备这次任务。

首先,是准备丹药。

苏清歌花了一整天,炼制了大量的疗伤丹、回气丹、破境丹等。这些丹药在关键时刻可以救命。

"夫君,这些丹药够了吗?"苏清歌问道。

"够了。"秦寒点头,"多谢。"

其次,是准备法宝。

秦寒检查了自己的苦海命运剑,这是他的本命法宝。此外,他还准备了一些符箓和阵盘。

"灵儿,你准备好了吗?"秦寒问道。

"准备好了!"赵灵儿拍了拍腰间的储物袋,"我带了很多灵果,路上饿了可以吃。"

秦寒和苏清歌都笑了。

"不是带吃的。"秦寒笑道,"是问你准备好了没有,能不能出发。"

"当然准备好了!"赵灵儿挺起胸膛,"我现在可是凝气期三重的高手!"

"凝气期三重..."秦寒摇了摇头,"算了,至少你能感知危险。"

一切准备就绪后,三人踏上了前往青州域北部的路。

青州域北部是一片荒芜的地区,人烟稀少,魔兽横行。那里常年被迷雾笼罩,阴森恐怖。

"听说北部以前是一个繁华的地区。"苏清歌说道,"但三百年前,一场突如其来的灾难降临,北部瞬间变成了一片废墟。从那以后,那里就成了魔兽的天堂。"

"什么灾难?"赵灵儿好奇地问。

"不知道。"苏清歌摇头,"古籍上没有详细记载,只说是一场'天灾'。"

"天灾..."秦寒喃喃自语,"或许,这次任务能找到一些线索。"

三人继续前行,朝着青州域北部飞去。

金丹期的飞行速度很快,一个时辰就能飞行千里。

但北部实在太远了,即使是全速飞行,也需要整整一天。

"我们找个地方休息一下吧。"苏清歌说道,"灵儿飞了一整天,应该累了。"

"我不累!"赵灵儿嘴硬道,但眼皮已经在打架了。

"休息吧。"秦寒笑道,"明天还要继续赶路。"

三人在一个小镇上落下,找了一家客栈住下。

晚饭后,秦寒独自站在客栈的屋顶上,看着北方的天空。

那里,乌云密布,雷电交加。

"北部..."秦寒喃喃自语,"那里到底隐藏着什么秘密?"

他隐隐感觉到,这次任务不会那么简单。

但无论前方有什么,他都绝不会退缩。

因为,他已经不是当年的那个弱小少年了。

现在的他,是金丹后期的修士!

他有足够的实力,去面对任何挑战!

# 第一百零三章 魔兽动乱

时间:接取任务后次日
地点:青州域北部
人物:秦寒、苏清歌、赵灵儿、魔兽、村民

次日清晨,三人继续赶路。

越往北走,周围的景色越发荒凉。原本郁郁葱葱的森林逐渐消失,取而代之的是枯黄的草木和干涸的河床。空气中弥漫着一股腐朽的气息,让人感到不适。

"这里...真的好荒凉。"苏清歌皱眉道,"完全不像我之前听说过的青州域北部。"

"听说三百年前,北部是一个繁华的地区。"秦寒说道,"人口数百万,城池数十座。但一场灾难后,一切都变了。"

"什么灾难能让整个地区变成这样?"赵灵儿好奇地问。

"不知道。"秦寒摇头,"古籍上没有详细记载,只说是'天灾'。但我觉得,未必那么简单。"

就在这时,赵灵儿突然停了下来,耳朵竖了起来。

"师兄,等等!"她紧张地说道,"我听到...哭声!"

"哭声?"秦寒和苏清歌都是一愣。

三人降落下来,朝着声音的方向飞去。

很快,他们看到了一个村庄。

或者说,曾经是村庄的地方。

现在这里只剩下一片废墟。房屋倒塌,墙壁破碎,到处都是烧焦的痕迹。废墟中,几个幸存者正在哭泣,为死去的亲人哀悼。

"太惨了..."苏清歌眼中含泪,"这些魔兽,简直没有人性!"

秦寒走到一位老妇人身边,蹲下身:"老人家,这里发生了什么?"

老妇人抬起头,满是皱纹的脸上写满了悲伤和恐惧:"是魔兽...成群结队的魔兽...它们突然出现,见人就杀,见屋就烧...我们拼命逃跑,但还是死了这么多人..."

"什么时候的事?"秦寒问。

"三天前..."老妇人说,"这些魔兽每隔几天就会出现一次,每次都会杀死很多人...我们根本无处可逃..."

"它们从哪里来?"

"北边..."老妇人颤抖的手指向北方,"它们是从北边的深山里来的...那里有一个巨大的山洞,是它们的巢穴..."

秦寒站起身,眼中闪过一丝杀意。

"师兄,怎么了?"赵灵儿问。

"这些魔兽必须消灭。"秦寒冷冷地说,"否则,还会有更多的人死去。"

就在这时,远处传来了一阵咆哮声。

"吼——!"

声音震天动地,让地面都微微颤抖。

"魔兽来了!"老妇人惊恐地喊道,"快跑!快跑啊!"

村民们纷纷逃窜,但秦寒却没有动。

"你们带村民先走。"秦寒对苏清歌和赵灵儿说道,"我来对付这些魔兽。"

"夫君,我和你一起!"苏清歌坚定地说。
