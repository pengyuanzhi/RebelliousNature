# 第一百四十一章 父子深入交谈

时间:父子团聚后
地点:玄天宗后山洞府
人物:秦寒、父亲

父子二人在洞府中坐下,秦寒给父亲泡了一杯茶。

这是他在万宝大会上买的"清心茶",能够平静心神,对于修炼很有帮助。父亲接过茶杯,轻轻抿了一口,脸上露出欣慰的笑容。

"寒儿,这些年苦了你了。"父亲感慨道,"一个人在中界打拼,肯定不容易吧?"

"还行。"秦寒笑道,"中界虽然危险,但机会也多。而且我遇到了好师父,还有很多朋友。"

"朋友?"父亲眼中闪过一丝好奇,"都是些什么样的人?"

"有一个叫慕容雪的师姐,是师父的二弟子。"秦寒说,"她修炼冰属性功法,性格虽然高冷,但人很好。还有一个大师兄白羽,虽然有些傲气,但实力很强。"

"还有其他朋友吗?"父亲问。

"有啊。"秦寒笑道,"我在洞天试炼中认识了不少朋友,虽然来自不同势力,但大家关系都不错。"

父亲欣慰地点点头:"有朋友就好。修真之路太孤单,有朋友相伴会好很多。"

父子二人聊了很久,从修炼聊到生活,从过去聊到未来。秦寒给父亲讲了中界的繁华,讲了万宝大会的热闹,讲了洞天仙府的美丽。父亲听得津津有味,眼中闪烁着光芒。

"父亲,您在中界生活过吗?"秦寒好奇地问。

父亲沉默片刻,缓缓点头:"是的,当年我年轻的时候,也曾去过中界。"

"真的?"秦寒震惊了,"那您在中界做什么?"

"修炼,寻找突破的机会。"父亲苦笑,"可惜我天赋有限,在中界待了十年,也只是从筑基后期突破到金丹初期。后来遇到你母亲,就回到下界了。"

"母亲...也是修士吗?"秦寒问。这是他第一次询问母亲的事。

父亲眼中闪过一丝痛苦,但很快平复下来:"是的,你母亲也是修士。她是丹塔的天才,天生冰灵根,修炼速度极快。当年的她,已经是金丹中期了。"

"金丹中期..."秦寒震惊。母亲这么强?那为什么会死在妖兽手中?

父亲深吸一口气,缓缓讲述了当年的往事。

"那是三十年前的事了。"父亲陷入回忆,"当时我在中界修炼,遇到了你母亲。她是丹塔的天才弟子,天生冰灵根,二十岁就达到了金丹中期。整个丹塔,甚至整个中界,都认为她将来一定能突破元婴期,甚至化神期。"

"母亲这么厉害..."秦寒震撼了。

"是啊。"父亲苦笑,"当时的我,只是个小小的筑基后期,根本配不上她。但你母亲不在乎这些,她说修炼最重要的是心意相通,而不是修为高低。"

"后来...你们是怎么在一起的?"秦寒好奇地问。

"是一次试炼。"父亲眼中闪过一丝温柔,"当时中界开启了一个古老遗迹,无数年轻天才都去探索。你母亲也在其中,我也碰巧去了。遗迹中危险重重,我们两个都不小心落单了。"

"然后你们遇到了?"

"遇到了一头元婴初期的守护妖兽。"父亲说,"以我们当时的实力,根本不是对手。关键时刻,你母亲挡在我面前,受了重伤。"

父亲的声音有些哽咽:"我拼命护着她,逃出了遗迹。之后,我一直在照顾她。养伤的过程中,我们产生了感情。伤好后,她决定跟我回下界。"

"母亲为什么要放弃中界的生活?"秦寒不解。

"因为丹塔希望她嫁给一个元婴期的长老。"父亲说,"她不愿意,所以跟我私奔了。回到下界后,我们隐姓埋名,在一座小山村里生活。那时候的日子很苦,但很幸福。"

秦寒听着,心中五味杂陈。原来母亲为了爱情,放弃了在中界的一切。

"后来,你出生了。"父亲继续道,"你母亲高兴得哭了。她说,这是上天给我们的恩赐。你出生后,我们的生活更加幸福。你很聪明,三岁就能识字,五岁就能背诵简单的口诀。我们以为,这样的日子会一直持续下去。"

父亲的眼泪流了下来:"可惜...好景不长。你六岁那年,我外出寻找灵药,想给你母亲一个惊喜。没想到遇到了一头元婴期的妖兽..."

父亲泣不成声,再也说不下去了。

秦寒心中一酸,紧紧抱住父亲。原来母亲是为了救父亲而死的,难怪父亲这么多年来一直走不出来。

"父亲,我一定会为您和母亲讨回公道!"秦寒握紧拳头。

"不,不要去报仇。"父亲摇头,"那头妖兽早就死了,被丹塔的高手击杀。报仇没有意义,你要做的,是好好活下去,成为强者,这样你母亲在天之灵才会欣慰。"

"父亲..."秦寒眼眶湿润。

"好了,不说这些伤心事了。"父亲擦干眼泪,"你刚才说要突破元婴期?"

"是的。"秦寒点头,"我已经买了四阶元婴丹,这次回来就是想在下界突破。"

"元婴期..."父亲感慨,"当年的我,也曾梦想突破元婴期。可惜后来发生了那么多事,修为一直停滞不前。"

"父亲现在是什么修为?"秦寒问。

"金丹初期。"父亲说,"当年你母亲去世后,我心灰意冷,修为一直没有进步。这些年多亏玄天宗照顾,我才能慢慢走出来。"

"父亲,等您恢复心情,我们一起修炼。"秦寒说,"我相信您一定能突破到元婴期。"

"好!"父亲眼中闪过希望,"有你这个儿子,我这一生值了。"

父子二人聊到深夜,秦寒才回到自己的洞府。

走在玄天宗的小径上,秦寒抬头望向星空。中界的星空虽然明亮,但下界的星空更加亲切。这里是他的根,是他的家。

"母亲,您放心,我一定会好好活下去,成为强者。"秦寒在心中默默说道。

# 第一百四十二章 突破元婴期

时间:返回下界后
地点:玄天宗闭关室
人物:秦寒

秦寒在玄天宗的闭关室中,准备突破元婴期。

这是一间位于玄天宗后山的闭关室,灵气虽然不如中界浓郁,但胜在清净。秦寒盘膝坐在蒲团上,调整呼吸,让自己的心境达到最佳状态。

他从储物戒中取出四阶元婴丹,这是他在万宝大会上花重金购买的。丹药呈琥珀色,表面光滑如玉,散发着淡淡的药香。光是握在手中,就能感受到一股温暖的灵力流入体内。

"元婴期,我来了!"秦寒握紧拳头。

他深吸一口气,一口吞下元婴丹。

丹药入腹,瞬间化作庞大的灵力,在他体内爆发。这股灵力如同奔腾的江水,在他的经脉中疯狂冲撞。秦寒连忙运转《苦海经》,引导这些灵力冲击元婴期的瓶颈。

轰隆隆!

体内传来雷鸣般的声响,那是瓶颈在震动的声音。

一次,两次,三次...

秦寒一次次冲击瓶颈,但那层瓶颈极其坚固,难以突破。金丹期到元婴期,这是修真路上的一道巨大坎。无数修士卡在这个关口,终生无法寸进。

"还不够!"秦寒咬牙。

他开启命轮眼,双眼之中金色的纹路浮现。命轮眼的力量涌出,与体内的灵力融合,形成一股更强大的力量。

轰!

终于,在秦寒的全力冲击下,那层坚固的瓶颈开始出现裂痕。

体内的金丹开始碎裂,化作点点金光。这些金光在秦寒的丹田中汇聚,逐渐形成一个婴儿的形状。

这就是元婴!

金色婴儿逐渐凝实,它的五官与秦寒一模一样。它盘膝坐在丹田中,身上散发着淡淡的灵光。元婴虽小,但蕴含的力量却极其恐怖,比金丹期强大了数倍不止。

"成功了!"秦寒睁开眼睛,眼中闪过一丝喜色。

他成功突破到了元婴初期!

突破之后,秦寒感受到体内发生了翻天覆地的变化。

首先是神识。之前他的神识只能覆盖方圆几里,而现在,神识范围扩大了十倍不止,可以清晰地感知到方圆数十里内的一切动静。

其次是灵力。体内的灵力更加凝实,更加强大。同样的招式,现在的威力至少是金丹期的三倍。

最后是寿元。金丹期的寿元是一千年,而元婴期的寿元延长到了两千年。这意味着他有大把的时间去追求更高的境界。

"这就是元婴期..."秦寒感慨道。

元婴初期,这是修真路上的一个重要分水岭。达到元婴期后,神识可以离体,寿元延长到两千年,实力远超金丹期。

"终于突破了!"秦寒握紧拳头,"母亲在天之灵,看到我今天的成就,一定会很欣慰。"

他走出闭关室,看到父亲和宗主都在外面等候。

"寒儿,你突破了?"父亲激动地问。

"是的。"秦寒点头,"我现在是元婴初期了。"

"太好了!"父亲流下眼泪,"你母亲如果在天有灵,一定会为你骄傲!"

"父亲,我会继续努力修炼。"秦寒说,"将来一定要成为下界最强者,守护我想守护的一切。"

"好!"父亲点头,"我等你。"

秦寒看着父亲,心中充满温暖。从小到大,父亲一直是他最坚实的后盾。现在他终于有能力保护父亲了。

"父亲,以后我会常回来看您的。"秦寒说,"等我在中界稳定下来,就来接您过去。"

"好,好。"父亲点头,"你以修炼为重,不用担心我。"

秦寒告别父亲和宗主,准备继续他的修炼之路。

# 第一百四十三章 晋升长老

时间:突破元婴期后
地点:玄天宗,大殿
人物:秦寒、宗主、父亲、众长老

秦寒突破元婴期的消息,很快传遍了整个玄天宗。

作为下界二流宗门,玄天宗历史上出过的元婴期修士不超过十个。而秦寒不仅修炼到了元婴期,还如此年轻——不到三十岁就达到了元婴初期,这在下界简直是奇迹!

宗主立即召集所有长老,在大殿商议。

"诸位,秦寒已经突破到元婴期。"宗主说,"他的实力,已经不在我们之下。我提议,晋升秦寒为我宗长老,诸位意下如何?"

此言一出,众长老纷纷议论。

"我同意!"大长老第一个表态,"秦寒的实力毋庸置疑,而且他为人正直,对我宗忠心耿耿,完全有资格成为长老!"

"我也同意。"二长老点头,"而且他还是秦大江的独子,可以说是我们看着长大的。"

"没错,秦寒天赋异禀,将来必成大器。"

所有长老都没有异议。

秦寒站在大殿中央,心中有些激动。虽然他在中界已经是洞天真人的弟子,但在下界,能够成为玄天宗长老,依然是一种荣誉。

"秦寒,从今日起,你就是我玄天宗长老。"宗主郑重地说,"你可以享受长老的一切待遇,包括修炼资源、宗门秘境、洞府等。"

"多谢宗主!"秦寒拱手。

"不,应该叫你秦长老!"宗主笑道。

众长老也都笑了。

"秦长老,恭喜你!"

"恭喜秦长老!"

秦寒一一回礼。

父亲站在一旁,眼中满是骄傲。他的儿子,不仅修炼到了元婴期,还成了玄天宗长老,这是何等的荣耀!

"寒儿,你终于出人头地了。"父亲激动地说。

"父亲,这还要感谢宗主和诸位长老一直以来的照顾。"秦寒说。

"哪里,是你自己努力。"宗主摆手,"对了,秦长老,长老洞府在后山,你可以选一处作为自己的修炼之地。另外,长老可以进入宗门秘境修炼,那里灵气比外界浓郁十倍。"

"多谢宗主。"秦寒点头。

成为长老后,秦寒在宗门的地位大大提升。以前他还需要向长老行礼,现在,他已经是和长老平起平坐的存在了。

不过,秦寒并没有因此骄傲自满。他深知,修真之路漫漫,元婴期只是其中的一个阶段。要成为真正的强者,还有很长的路要走。

"父亲,您以后可以搬到长老洞府附近。"秦寒对父亲说,"这样我也能照顾您。"

"好,好。"父亲连连点头。

秦寒又去见了一直照顾自己的师叔们,感谢他们的教导。师叔们看到秦寒如今成就,也都为他高兴。

"秦寒,你以后有什么打算?"师叔问。

"我打算先在下界待一段时间,巩固修为。"秦寒说,"然后返回中界,继续修炼。"

"嗯,这是个好主意。"师叔点头,"元婴期刚突破,需要时间巩固。否则根基不稳,将来难以寸进。"

"我会小心的。"秦寒说。

成为长老后,秦寒开始享受长老待遇。他选了一处靠近父亲的长老洞府,环境清幽,灵气浓郁。每天除了修炼,就是陪伴父亲。

这样的日子,过得平静而充实。

# 第一百四十四章 苏清歌

时间:晋升长老后数日
地点:太华宗,青州城
人物:秦寒、苏清歌、苏父

成为玄天宗长老后的第二天,秦寒决定去太华宗一趟。

虽然他和苏清歌已经结为道侣,但由于一直在中界试炼,两人已经很久没有见面了。这次返回下界,秦寒想先去见见她。

秦寒祭起飞剑,化作一道流光,向太华宗方向飞去。

太华宗位于青州域东部,距离玄天宗有数千里之遥。如果是金丹期修士,飞行需要大半个月。但现在秦寒已是元婴期,速度提升了数倍,只用了一天就到了。

太华宗,作为下界一流宗门,气势恢宏。秦寒出示长老令后,顺利进入宗门。

"秦长老,欢迎莅临我宗!"太华宗弟子恭敬地说。

秦寒点点头,直奔丹塔。

苏清歌作为丹塔天才,在太华宗地位很高。她的洞府在丹塔后山,环境清幽。

"清歌!"秦寒喊道。

洞府中走出一位少女,正是苏清歌。她穿着一身素白色长裙,长发披肩,容貌绝美。看到秦寒,她眼中闪过惊喜。

"秦寒哥哥!"苏清歌扑进秦寒怀里。

"清歌,我想你了。"秦寒抱着她。

"我也想你。"苏清歌说,"你在中界还好吗?"

"很好,我突破到元婴期了。"秦寒说。

"真的?太好了!"苏清歌为他高兴,"那你什么时候返回中界?"

"不急,我这次回来,主要是想多陪陪你。"秦寒说,"另外,我父亲还在玄天宗,我也要照顾他。"

"嗯,我理解。"苏清歌点头,"对了,我的修为也突破到金丹后期了!"

"恭喜你!"秦寒说,"你的修炼速度很快,将来一定能突破到元婴期。"

"那是当然,有你在,我怎么会落后?"苏清歌笑道。

两人聊了很久,从修炼到生活,从过去到未来。苏清歌告诉秦寒,她最近在研究一种新的丹药,如果能成功,可以大幅提升修炼效率。

"太厉害了!"秦寒赞叹,"清歌,你在丹道上的天赋真是令人惊叹。"

"哪里,是你鼓励得好。"苏清歌害羞地说。

就在这时,一位中年男子走了过来。正是苏清歌的父亲,丹塔塔主。

"秦寒,你回来了?"苏父笑道。

"是的,苏叔叔。"秦寒拱手。

"听说你突破到元婴期了,了不起!"苏父拍拍秦寒的肩膀,"你现在可是下界顶尖强者了。"

"苏叔叔过奖了。"秦寒谦虚地说。

"不是我过奖,是事实。"苏父说,"秦寒,我有个不情之请。"

"苏叔叔请说。"

"清歌的体质特殊,修炼起来比常人困难。虽然她现在已经金丹后期,但想要突破到元婴期,还需要机缘。你将来在中界,如果能找到改善体质的宝物,还请多多帮助。"苏父认真地说。

"苏叔叔放心,清歌是我的道侣,我一定会帮助她的。"秦寒郑重地说。

"好,有你这句话,我就放心了。"苏父点头。

秦寒在太华宗待了三天,这三天里,他和苏清歌形影不离,一起修炼,一起谈心,感情更加深厚。

第三天,秦寒准备返回玄天宗。

"清歌,我要回去了。"秦寒说,"父亲还在等我,而且我也要巩固修为。"

"嗯,你回去吧。"苏清歌说,"我会继续努力修炼的,争取早日突破到元婴期,然后去中界找你。"

"好,我在中界等你。"秦寒说,"对了,等我元婴期稳固后,就来接你过去。"

"好,一言为定!"

两人依依惜别。

# 第一百四十五章 发现父亲的旧伤

时间:返回玄天宗后
地点:玄天宗,长老洞府
人物:秦寒、父亲

秦寒返回玄天宗,准备继续闭关巩固元婴期修为。

就在他准备进入修炼室时,突然注意到父亲的脸色有些不对。

"父亲,您怎么了?"秦寒关切地问。

"没,没什么。"父亲摆手,"可能是最近有些累了。"

秦寒皱眉。父亲的表情虽然在掩饰,但他还是看出了端倪。父亲的气息有些紊乱,体内的灵力运行也不太顺畅。

"父亲,让我看看。"秦寒说。

"真的没事。"父亲还想推辞。

"父亲!"秦寒语气严肃起来。

父亲见瞒不过去,只好点头:"好吧,确实有点不舒服。"

秦寒立即开启命轮眼。金色的纹路在眼中浮现,他开始仔细检查父亲的身体。

很快,他就发现了问题。

父亲体内有旧伤!

这些旧伤应该是早年留下的,一直没有得到妥善治疗。旧伤位于父亲的心脉和丹田附近,虽然不致命,但会严重影响修炼和健康。

"父亲,您怎么不告诉我?"秦寒心疼地说。

"这点小伤,不碍事的。"父亲笑着说。

"这可不是小伤!"秦寒严肃地说,"您的旧伤位于心脉和丹田,如果不及时治疗,会影响修炼,甚至可能危及生命!"

"没那么严重..."父亲还在推辞。

"父亲!"秦寒打断他,"您是我唯一的亲人,我不能眼睁睁看着您受伤而不治疗。"

父亲沉默了。

良久,他才叹气:"寒儿,其实这些伤,是当年去上界寻找你母亲时留下的。"

"母亲?"秦寒心中一震。

"是的。"父亲说,"当年你母亲去世后,我不甘心,听说上界可能有起死回生的办法,就冒着生命危险去了。但在上界,我实力太弱,受了重伤,只能狼狈逃回。这些伤,就是那时留下的。"

秦寒心中五味杂陈。

原来,父亲一直都没有忘记母亲。这些年来,父亲虽然表面上看起来坚强,但内心深处,始终怀着对母亲的愧疚和思念。

"父亲,您为什么不早告诉我?"秦寒问。

"告诉你又有什么用?"父亲苦笑,"你那时还小,我怎么能让你担心?"

秦寒眼眶湿润。

父亲就是这样,永远把苦自己扛,把好留给他。

"父亲,我一定会治好您的!"秦寒握紧拳头。

"寒儿,不必了。"父亲摇头,"这些伤跟了我几十年,早就习惯了。你现在是元婴期修士,应该以修炼为重,不要在我身上浪费时间。"

"父亲!"秦寒坚定地说,"您是我父亲,为我付出了一切。现在我有能力了,怎么能不治疗您?您放心,我在万宝大会上购买了一些疗伤丹药,应该能治好您的旧伤。"

父亲还想推辞,但看到秦寒坚定的眼神,最终点了点头:"好吧,那就麻烦你了。"

"父亲,您是我父亲,说什么麻烦不麻烦。"秦寒说。

他立即取出在万宝大会购买的疗伤丹药。这是一种四阶丹药,名为"复元丹",专门治疗旧伤,价值连城。

"父亲,您服下这枚丹药,我帮您疗伤。"秦寒说。

父亲点头,接过丹药服下。

秦寒伸出双手,按在父亲的背上。体内的元婴之力缓缓流出,进入父亲体内,帮助化解药力,驱除旧伤。

这个过程持续了整整一个时辰。

一个时辰后,秦寒收回双手,脸上露出疲惫的神色。

"父亲,您的旧伤已经治好了七成。"秦寒说,"剩下的,需要慢慢调养。"

"好,好。"父亲激动地说,"寒儿,谢谢你!"

"父亲,您是我父亲,为什么要说谢谢?"秦寒笑了。

父亲也笑了。

# 第一百四十六章 父亲痊愈

时间:疗伤后数日
地点:玄天宗,长老洞府
人物:秦寒、父亲

经过几天的调养,父亲的旧伤基本痊愈了。

这一天,秦寒再次检查父亲的身体,发现心脉和丹田的旧伤已经完全愈合,体内灵力运行也恢复了正常。

"父亲,您的伤完全好了!"秦寒高兴地说。

父亲感受了一下体内的灵力,确实顺畅了许多。以前的种种不适——胸闷、气短、灵力凝滞——全都消失了。

"太好了!"父亲激动地说,"寒儿,真的谢谢你!"

"父亲,您是我父亲,不用谢。"秦寒笑道。

"不,这次真的要好好谢谢你。"父亲认真地说,"这些旧伤跟了我几十年,让我吃尽了苦头。如果不是你,我恐怕这辈子都好不了了。"

"父亲,您为我付出了一切,我为您治疗一点伤,又算得了什么?"秦寒说。

父亲眼中闪过泪光。

他想起当年秦寒出生时,家中一贫如洗。为了给秦寒买米,他每天天不亮就上山砍柴,然后挑到镇上去卖,风里来雨里去,从不抱怨。

后来母亲去世,他更是承担起所有责任,既要照顾秦寒,又要维持生计。那些年,他真的很累,但从未放弃。

"寒儿,你长大了。"父亲感慨地说,"不仅修炼到了元婴期,还这么孝顺。你母亲如果在天有灵,一定会很欣慰。"

"父亲,您别这么说。"秦寒说,"如果不是您,我不可能有今天。您是我生命中最重要的人。"

"好,好。"父亲连连点头,"对了,寒儿,你的修为已经巩固得怎么样了?"

"差不多了。"秦寒说,"我准备继续闭关,争取突破到元婴中期。"

"好,你以修炼为重,不用管我。"父亲说,"我现在身体好了,可以照顾自己。"

"父亲,我会常来看您的。"秦寒说。

"嗯,去吧。"父亲点头。
