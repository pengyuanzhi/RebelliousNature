# 第一百三十五章 金丹后期的巩固

时间：万宝大会前夕
地点：洞天仙府修炼室
人物：秦寒

---

洞天仙府，修炼室。

这是一间位于洞天仙府深处的修炼密室，四周墙壁上镶嵌着数百颗月光石，散发着柔和的光芒。修炼室中央，是一个由寒玉打造的蒲团，寒玉散发着丝丝凉意，能够帮助修炼者平复心境。

修炼室的顶部，布置着一个小型的聚灵阵。这个阵法是洞天真人亲自布置的，能够将周围数十里的灵气聚集到修炼室内，使灵气浓度达到外界的三倍以上。

秦寒盘膝坐在寒玉蒲团上，双目微阖，心神完全沉浸在修炼之中。

万宝大会还有一个月就要开启，这一个月内，他需要将金丹后期的修为巩固到圆满状态。

"金丹后期..."秦寒心中默默感受着体内的变化。

突破到金丹后期后，他感觉自己的实力有了质的飞跃。首先是灵力的深厚程度，比金丹中期时增强了数倍。以前只能施展几十次强力招式，现在可以施展数百次而不枯竭。

其次是神识的增强。以前只能感知方圆百里的动静，现在可以感知方圆千里。而且神识的敏锐度也提升了，可以捕捉到空气中微小的灵气波动。

最重要的是，他对天地法则的感悟加深了。以前只能模糊地感觉到法则的存在，现在可以清晰地看到一些法则的流动。比如火焰法则、雷电法则、空间法则等等。

"这就是金丹后期的力量吗..."秦寒心中感叹。

在修真界，金丹期是一个分水岭。金丹期之前的修士，只能算是修真界的底层。而金丹期修士，已经算是中坚力量。

金丹初期，可以担任一宗的长老。
金丹中期，可以成为一方的强者。
金丹后期，在整个青州域都是顶级存在，即使是宗门掌主也要礼让三分。

而在中界，金丹后期虽然不算顶级，但也算是中等水平。只要不遇到元婴期的高手，金丹后期修士都可以自保。

"不过，在中界，金丹后期只是起步。"秦寒暗道，"想要在中界立足，甚至将来保护想保护的人，我必须突破到元婴期。"

秦寒深吸一口气，开始运转《苦海经》。

《苦海经》是道阶心法，修炼难度极高，但威力也极大。它以"苦"为核心，将修炼者的痛苦转化为力量。身体之苦、心灵之苦、命运之苦，所有的痛苦都会成为修炼的助力。

随着《苦海经》的运转，秦寒体内的金丹开始缓缓旋转。

金丹只有拇指大小，通体金色，表面隐约可见一些纹路。那些纹路是天地灵气凝聚而成的道韵，蕴含着玄奥的道理。

每一次金丹旋转，都会吸收大量的灵气。这些灵气经过金丹的炼化，会变得更加纯净、更加凝练。原本无色的灵气，经过金丹的炼化后，会变成淡淡的金色，其中蕴含着强大的力量。

"继续巩固..."秦寒心中默念。

他小心翼翼地控制着灵力，让它们缓缓流过金丹的表面。每一次流转，金丹都会变得更加凝实。

金丹的凝实程度，直接决定了修为的强弱。同样都是金丹后期，有的修士金丹凝实，实力是普通金丹后期的两倍。而有的修士金丹虚浮，虽然是金丹后期，但实力只相当于金丹中期。

秦寒的金丹，是极品金丹。从凝丹的那一刻起，他的金丹就达到了九成九的纯净度。这是传说中的极品金丹，只有那些绝世天才才能凝聚出来。

"极品金丹..."秦寒心中庆幸。

正是因为有极品金丹，他的修炼速度才会这么快。同样是吸收灵气，他的金丹能够吸收更多的灵气，炼化的效率也更高。

而且，极品金丹还有一个巨大的优势——突破元婴期的成功率更高。

普通修士，从金丹后期突破到元婴期，成功率只有一成。而极品金丹的修士，成功率可以达到五成，甚至更高。

"等我突破到元婴期，实力会有更大的飞跃。"秦寒期待着。

元婴期修士，体内已经凝聚出元神。元神可以离体神游，可以施展神通，可以借助天地之力。实力是金丹期的十倍以上！

就在这时，秦寒感觉到命轮眼有了反应。

命轮眼是他的特殊能力，能够看破因果、预知危险、改变命运。随着修为的提升，命轮眼的能力也在不断增强。

"现在的命轮眼，可以看到什么..."秦寒想。

他运转命轮眼，眼前的世界顿时发生了变化。

他看到了无数条线——那是因果线。

每一条因果线，都连接着两个人或物。比如他和苏清歌之间，有一条粗大的因果线，说明两人有着深厚的缘分。他和赵灵儿之间，也有一条因果线。

而那些陌生人之间，因果线很细，甚至看不到。

"因果..."秦寒心中感悟。

命轮眼不仅可以看到因果，还可以改变因果。比如，如果一个人原本要死，命轮眼可以看到他的死亡因果线。然后，通过改变一些事情，可以斩断这条因果线，从而改变这个人的命运。

"不过，改变因果需要付出代价。"秦寒想。

每一次使用命轮眼改变因果，都会消耗他的寿元。而且，改变的事情越重大，消耗的寿元越多。

"不到万不得已，不能随意使用命轮眼改变因果。"秦寒告诫自己。

除了看因果，命轮眼还可以预知危险。

秦寒运转命轮眼，看向修炼室的门口。

突然，他看到了一幕——修炼室的门被推开，慕容雪走了进来。

几息之后，修炼室的门真的被推开了！

"秦师兄，你出来一下，我有话对你说。"慕容雪的声音从门口传来。

秦寒睁开眼睛，嘴角露出一丝笑意。

命轮眼的预知能力果然准确。他刚刚看到的画面，真的发生了。

"怎么了？"秦寒站起身，走出修炼室。

慕容雪站在修炼室门口，脸上带着一丝严肃。

"师父让我叫你过去，说是有重要的事情。"慕容雪说道。

"好，我这就去。"秦寒点头。

两人一起朝着洞天真人的洞府走去。

路上，慕容雪忽然说道："秦师兄，你的修为...提升了？"

秦寒一愣，然后点头："是啊，刚刚巩固了金丹后期。"

"难怪..."慕容雪感叹，"我感觉你身上的气息比之前强了不少。金丹后期...我也要加油了。"

慕容雪现在是金丹后期修为，但还没有达到圆满。她本来以为自己比秦寒强，没想到秦寒这么快就追上来了。

"你可以的。"秦寒鼓励道，"你的天赋很高，只要努力，一定能突破到元婴期。"

"谢谢秦师兄。"慕容雪笑了，眼中闪过一丝欣赏。

她原本对秦寒有些不服气，但经过这段时间的相处，她已经认可了秦寒的实力和人品。秦寒不仅天赋高，而且勤奋、谦虚、重情重义，是一个值得尊敬的对手。

两人来到洞天真人的洞府。

洞府很宽敞，布置得很雅致。洞府中央，洞天真人正在品茶。

"师父。"两人行礼。

"坐。"洞天真人点头。

秦寒和慕容雪坐下后，洞天真人说道："叫你们来，是为了万宝大会的事情。"

"万宝大会？"秦寒好奇。

"万宝大会是中界十年一度的盛事。"洞天真人解释道，"由万宝楼举办，届时整个中界的修士都会聚集到万宝城，交易各种宝物。"

"听起来很热闹。"秦寒说。

"不只是热闹。"洞天真人认真地说，"万宝大会上，经常会有一些稀世珍宝出现。你这次去，主要是为了购买几件关键的东西。"

"什么东西？"

"第一，辅助突破元婴期的丹药。"洞天真人说，"这种丹药极其珍贵，只有在万宝大会上才能买到。"

秦寒点头。突破元婴期是当务之急，如果有辅助丹药，成功率会更高。

"第二，一件趁手的法宝。"洞天真人继续说，"苦海命运剑虽然强大，但毕竟是本命神器，不能随意使用。你需要一件普通的法宝来应对日常战斗。"

"明白。"秦寒想。确实，苦海命运剑太珍贵了，而且每次使用都会消耗他的精血。需要一件普通的法宝来应对普通的战斗。

"第三，阵法相关的资源。"洞天真人说，"你对阵法研究不深，但阵法在关键时刻能救命。如果能买到几件阵法法宝，实力会大增。"

"弟子记住了。"秦寒认真地点头。

"还有一个月的时间。"洞天真人说，"这一个月你好好巩固修为，争取在万宝大会前达到金丹后期圆满。届时为师会亲自带你们去。"

"是！"秦寒应道。

离开洞府后，秦寒回到修炼室，继续修炼。

"金丹后期圆满..."秦寒暗道，"还有一个月时间，应该够了。"

他重新盘膝坐下，运转《苦海经》。

体内的金丹缓缓旋转，不断吸收周围的灵气。每一次吸收，金丹都会变得更加凝实。

秦寒能够感觉到，距离金丹后期圆满，只差一步了。

"万宝大会，我会展现实力的。"秦寒握紧拳头。
