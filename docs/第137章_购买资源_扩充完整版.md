# 第一百三十七章 购买资源

时间：万宝大会第一天
地点：万宝广场交易区
人物：秦寒、摊主、黑袍修士

---

万宝大会第一天，交易区。

万宝广场位于万宝楼的正前方，是一个巨大的广场，足有千丈方圆。广场上密密麻麻地摆满了摊位，每个摊位前都围着不少修士。

洞天真人对秦寒三人说道："你们三个自由活动吧，去买自己需要的东西。为师要去见几个老朋友，晚上再汇合。"

"是，师父。"三人齐声应道。

洞天真人离开后，慕容雪说道："秦师兄，我要去剑楼看看，那里有各种剑类法宝。"

"我要去丹药区。"白羽说，"想买几颗辅助修炼的丹药。"

"好，那我们分头行动。"秦寒点头，"晚上在这里汇合。"

三人分头离去。

秦寒独自在广场上逛着。他对法宝不感兴趣，苦海命运剑已经足够强大了。他主要想找三样东西：元婴丹、阵法资源、以及一些稀有的灵草。

万宝广场太大了，摊位数以千计。秦寒逛了一个时辰，才逛了不到十分之一。

这些摊位中，什么都有卖的：

有的摊位卖丹药，从筑基期的淬体丹到元婴期的元婴丹，应有尽有。但价格都很贵，最便宜的淬体丹也要十块中品灵石一颗。而元婴丹，最便宜的也要上万上品灵石。

有的摊位卖法宝，刀枪剑戟，斧钺钩叉，每一件都散发着灵光。但这些法宝大多是玄阶和地阶，天阶法宝很少见。

还有的摊位卖灵草、灵石、功法秘籍、符箓阵旗...琳琅满目，应有尽有。

"这万宝大会果然名不虚传。"秦寒心中感叹。

不过，价格也很贵。以他现在的身家，根本买不起什么好东西。洞天真人给了他五万上品灵石，听起来很多，但在万宝大会，这点钱根本不够花。

逛了一会儿，秦寒在一个摊位前停下了脚步。

这个摊位不大，但摆放的东西却很精致。摊主是一个老者，须发皆白，看起来平平无奇，只有筑基期修为。但秦寒能够感觉到，这个老者不简单——他的气息内敛，显然是隐藏了修为。

"小友，看中什么了？"老者笑眯眯地问，声音温和，给人一种亲切的感觉。

秦寒的目光落在摊位上的一枚丹药上。

那枚丹药只有龙眼大小，通体晶莹，散发着璀璨的金色光芒。丹药表面，隐约可见一些纹路，那些纹路流转不定，仿佛有生命一般。

更神奇的是，丹药周围竟然形成了一个小型的灵气漩涡！浓郁的地灵气被丹药吸收，然后又化作更加纯净的灵气散发出来。

"这是..."秦寒心中一凛。

他修炼炼丹术，对丹药有一定的了解。能够自动吸收灵气、然后化作纯净灵气散发出来的丹药，绝对是极品丹药！

"小友好眼力。"老者赞许道，"这是四阶元婴丹，而且是一枚极品元婴丹。"

"极品元婴丹？"秦寒惊讶道。

四阶丹药，是给元婴期修士使用的丹药。而元婴丹，是所有四阶丹药中最难炼制的，成功率不足一成。即使是一流炼丹宗师，炼制十次，能成功一次就不错了。

而极品元婴丹，更是凤毛麟角。极品丹药比普通丹药的功效强数倍，而且没有丹毒，可以放心服用。

"不错。"老者点头，"这枚元婴丹，是为师花了三年时间，集齐了百种灵草，历经九九八十一次失败，才炼制成功的。服用后，可以增加三成突破元婴期的几率。而且还能稳固元婴，让元婴更加凝实。"

秦寒心中一动：三成几率...这太珍贵了！

要知道，从金丹期突破到元婴期，普通修士的几率只有一成。而极品金丹的修士，几率可以达到五成。如果再服用这枚极品元婴丹，几率就能达到八成！

八成的成功率...这意味着，只要不遇到意外，基本上就能突破成功！

"不过..."老者话锋一转，"这枚元婴丹的价格不菲，需要一万上品灵石。"

一万上品灵石！

秦寒倒吸一口冷气。

上品灵石和中品灵石的兑换比例是1:100，一万上品灵石就是一百万中品灵石！对于金丹期修士来说，这是一笔巨款。即使是金丹后期的高手，全部身家可能也就几万上品灵石。

"怎么，小友嫌贵？"老者笑道。

"不嫌贵。"秦寒摇头，"只是...一万上品灵石，我现在的积蓄不够。"

他现在有三万上品灵石，是洞天真人给他的。但如果花一万买这枚丹药，就只剩下两万了。而后续还需要买阵法资源、灵草等等，钱可能不够。

"无妨。"老者说，"你可以先付定金，我为你保留三天。三天内如果你筹够了钱，就来取。如果筹不够，定金双倍返还。"

"多谢前辈。"秦寒心中感激。这个老者，虽然看起来是商人，但很讲义气。

就在秦寒准备付定金时，一个冰冷的声音从身后传来：

"这枚元婴丹，我要了。"

秦寒转头，看到一个身穿黑袍的年轻人走了过来。

这人看起来约莫三十岁，面容阴鸷，双目赤红，散发着浓烈的煞气。他的修为达到了金丹后期巅峰，只差一步就能突破到圆满。而且，从他身上散发的气息来看，他修炼的是魔功。

"这位道友，是我先看中的。"秦寒淡淡道。

"那又如何？"黑袍人冷笑，"在中界，拳头大就是道理。我要它，它就是我的。"

周围围观的修士纷纷议论：

"是血煞门的弟子..."
"血煞门是魔道大宗门，在中界很有势力..."
"这个黑袍人修为这么高，恐怕是血煞门的核心弟子..."

秦寒心中了然。血煞门，他在下界时就听说过。那是魔道大宗门，实力强大，而且行事狠辣。没想到在中界也能遇到血煞门的弟子。

"道友，既然我先看中，这就是我的了。"秦寒不卑不亢地说道，"这是规矩。"

"规矩？"黑袍人哈哈大笑，"在中界，我的话就是规矩！"

话音刚落，他猛地一掌拍向摊位！

轰！

恐怖的掌风席卷而出，直接将摊位上的物品吹飞！那枚元婴丹也被卷起，朝着黑袍人飞去！

"你敢！"老者大怒。

但他只是筑基期的修为，根本无法阻止黑袍人的行动。

就在元婴丹即将落入黑袍人之手时，一道金光闪过！

秦寒出手了！

他身形一闪，瞬间出现在元婴丹前方，一把抓住了丹药。然后，他借力一蹬，身形向后飘退，稳稳落在地上。

"好快的速度！"黑袍人眼中闪过一丝惊讶，"看来你有点本事。"

"让开。"秦寒冷冷道，"这枚丹药，我要了。"

"你找死！"黑袍人大怒，身上煞气爆发！

周围数十丈内，温度骤降！一股恐怖的威压席卷而出，让周围的金丹期修士纷纷后退。

"这是金丹后期巅峰的威压..."
"而且，他修炼的魔功非常强大..."
"这个年轻人要倒大霉了..."

秦寒却面不改色，他同样释放出威压，与黑袍人对抗！

两股威压在空中碰撞，发出无声的轰鸣！

周围的修士更加惊讶了：

"这个年轻人...竟然能够硬抗血煞门弟子的威压？"
"他的修为...也是金丹后期巅峰！"
"而且，他的金丹气息很纯净，应该是极品金丹！"

"极品金丹？！"黑袍人脸色一变，"下界来的蝼蚁，竟然是极品金丹...有趣，有趣！"

他舔了舔嘴唇，眼中露出贪婪的光芒："极品金丹的金丹，听说可以用来炼制邪术，增强修为...把你抓回去，我的修为一定能更进一步！"

秦寒眼中闪过一丝寒芒："如果你想死，我可以成全你。"

"哈哈！口气不小！"黑袍人大笑，"那就来试试，看是你的嘴硬，还是我的刀硬！"

话音未落，他再次出手了！

这一次，他直接祭出了法宝——一把血红色的长刀！

长刀迎风暴涨，瞬间变成百丈长短，带着浓烈的煞气，朝着秦寒砍下！

"血煞斩！"黑袍人怒吼。

这一刀，蕴含了他十成的修为，威力足以秒杀普通的金丹后期修士！

周围的修士纷纷惊呼：

"是血煞门的血煞刀！"
"这一刀威力太强了..."
"那个年轻人要完了..."

就在长刀即将砍中秦寒时，秦寒动了！

他拔出了苦海命运剑！

锵！

一道金色的剑光冲天而起，直接迎上了血红色的长刀！

当！

两刀相撞，爆发出震耳欲聋的轰鸣！

周围的修士纷纷后退，以免被余波伤到。

烟尘散去，秦寒手持苦海命运剑，稳稳站在原地。而黑袍人的长刀，竟然被挡住了！

"什么？！"黑袍人震惊了，"你的剑...什么来头？"

"斩你，足矣。"秦寒冷冷道。
