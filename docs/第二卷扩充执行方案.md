## 四、扩充建议

### 写作要点

1. **环境描写**
   - 使用感官细节(视觉、听觉、嗅觉、触觉)
   - 营造氛围(恐怖、紧张、温馨等)
   - 与情节呼应

2. **人物刻画**
   - 对话要符合人物性格
   - 心理描写要深刻
   - 动作描写要详细

3. **战斗描写**
   - 招式名称要具体
   - 战斗过程要详细
   - 伤害后果要真实

4. **情感描写**
   - 情感发展要自然
   - 避免突兀
   - 细腻感人

### 扩充技巧

1. **对话扩充**
   - 原文:"他们商量了一下。"
   - 扩充:详细描写商量的过程,每个人的发言,讨论的细节等

2. **动作分解**
   - 原文:"他打出一拳。"
   - 扩充:描写站姿、蓄力、出拳、命中、后退等完整过程

3. **心理活动**
   - 原文:"他决定去。"
   - 扩充:描写他的犹豫、思考、权衡利弊、最终下定决心

4. **环境烘托**
   - 原文:"天气很好。"
   - 扩充:描写蓝天白云、阳光明媚、鸟语花香等具体景象

## 七、快速扩充命令

由于扩充工作量巨大(需增加约180,000字),建议采用以下策略:

### 策略A: 批量扩充(推荐)
一次扩充一个完整的章节(如第103章),包含完整的3000+字内容。

**优点**: 章节完整,连贯性好
**缺点**: 单次工作量大

### 策略B: 分段扩充
将每章分为3-4个部分,每次扩充一个部分。

**优点**: 单次工作量小
**缺点**: 需要多次操作,容易失去连贯性

### 策略C: AI辅助扩充
使用AI工具辅助扩充,然后人工审核和修改。

**优点**: 速度快
**缺点**: 需要仔细审核质量和连贯性
