# 第二百零一章 突破化神

时间：虚空秘境中
地点：虚空秘境·祭坛
人物：秦寒、苏清歌、玄天太上

秦寒盘膝坐在祭坛上，准备突破化神期。

他吞服了之前积攒的所有化神丹。

丹药入口即化，化作恐怖的药力。

秦寒全力运转《化神诀》和《苦海经》，引导药力淬炼元婴。

元婴在药力的滋养下，开始发生变化。

它渐渐变得模糊，仿佛要融化一般。

"化形……"秦寒喃喃。

元婴开始化形，逐渐脱离肉身的束缚。

这是化神期的第一步——元婴化神。

突然！

恐怖的心劫降临！

这是化神期必须经历的心性考验。

心劫化作无数幻象，攻击秦寒的心智。

"母亲……"秦寒看到了母亲的身影。

"寒儿……"母亲温柔地呼唤。

秦寒心中一软，差点迷失。

"不……那不是真的……"

他用命轮眼看破幻象。

第三重——看破幻境！

幻象消散。

但心劫没有结束。

接下来，是父亲、婆婆、亲朋好友……

一个个幻象出现，试图击溃秦寒的心智。

但都被秦寒一一识破。

"我的道心，坚如磐石！"秦寒坚定道。

心劫见状，使出最后一招——诱惑！

幻象中，秦寒看到了自己成为至强者的场景。

他站在众生之巅，俯瞰天下，享受无尽的荣华富贵。

"这是你要的……"心劫诱惑道。

"不。"秦寒摇头，"我要的不是这些。"

"那你要什么？"

"我要的是——自由。"秦寒道，"不受命运束缚的自由。"

"命由天定？"他冷笑，"我命由我不由天！"

心劫终于消散。

"心劫，过！"秦寒睁开眼。

元婴彻底化神，化作一个独立的神识体！

"化神初期！"秦寒激动。

他的气息，比之前强了数倍。

神识可以离体千里，掌控周围的一切。

"恭喜。"玄天太上微笑，"成功突破化神期。"

"多谢前辈。"秦寒抱拳。

"不用客气。"玄天太上道，"你既然已经突破，就赶紧离开这里吧。"

"为什么？"

"虚空之心被我取走，虚空秘境即将崩塌。"玄天太上道。

"什么？！"秦寒大惊。

"快走！"

玄天太上带着秦寒和苏清歌，冲向虚空裂缝。

三人刚离开，虚空秘境就开始崩塌。

"好险……"苏清歌道。

"多谢前辈。"秦寒再次感谢。

"应该的。"玄天太上道。

三人返回下界。

# 第二百零二章 化神初成

时间：突破化神后
地点：下界·青州域
人物：秦寒、苏清歌

秦寒突破化神期的消息，很快传遍下界。

下界各派纷纷派人前来祝贺。

"恭喜秦寒突破化神期！"

"下界第一人！"

赞美之声不绝于耳。

秦寒却没有因此而骄傲。

他深知，化神期只是开始。

前方的路，还很长。

"清歌，"秦寒道，"我准备闭关一段时间，稳固修为。"

"好的。"苏清歌点头。

秦寒进入闭关室，开始稳固化神期的修为。

化神期与元婴期截然不同。

最大的区别，是神识可以离体。

秦寒尝试着让神识离体，探索周围的环境。

"好神奇……"他喃喃。

神识可以感知到一切，千里之外，尽在掌握。

"化神期，果然强大。"

三个月后。

秦寒的修为，稳定在化神初期中期。

而且，他对化神期的运用，也越来越熟练。

神识可以分化成多个分身，同时做不同的事情。

"分身术……"秦寒满意。

他创造了一个神识分身，派往各地侦查。

"分身，去吧。探查下界的情况。"

分身领命而去。

秦寒的本体，则继续修炼。

他参悟《化神诀》，学习更多的化神期神通。

化神期的神通，比元婴期强大太多。

比如——

神识攻击：可以直接攻击敌人的神魂。

空间法则：可以操纵空间，瞬移千里。

时间法则：可以短暂地减缓或加速时间。

"这些神通，都需要慢慢掌握。"秦寒道。

他开始逐一修炼。

半年后。

秦寒对化神期的神通，已经基本掌握。

"差不多了。"秦寒出关。

"你出关了？"苏清歌迎上来。

"嗯。"秦寒微笑，"现在我是化神初期中期，实力比之前强了太多。"

"太好了。"苏清歌欣喜。

"清歌，"秦寒道，"你也该突破了。"

"我？"苏清歌道，"我的修为只有元婴初期中期……"

"你该突破到元婴后期了。"秦寒道。

"可是……"

"别可是。"秦寒道，"我来帮你。"

他开始帮助苏清歌突破。

利用命轮眼改善她的体质，传授她更高深的功法。

一年后。

苏清歌突破到元婴后期初期！

"成功了！"苏清歌激动。

"多谢你。"她抱住秦寒。

"不用谢。"秦寒微笑。

"我们是一家人。"

两人相拥而坐，享受着难得的平静。

# 第二百零三章 新的威胁

时间：秦寒出关后
地点：下界·青州域·太华宗
人物：秦寒、玄天真人

秦寒出关后，立即前往太华宗见掌门。

"掌门。"秦寒道。

"秦寒，你出关了？"玄天真人惊喜。

"是的。"秦寒点头，"而且，我突破了化神期。"

"化神期？！"玄天真人大惊。

"没错。"秦寒道。

"太好了……"玄天真人激动，"有了你这化神期，下界就有保障了。"

"掌门，"秦寒道，"我还有一事要告诉你。"

"什么事？"

"我派出的神识分身，发现了新的威胁。"秦寒道。

"新的威胁？"玄天真人凝重。

"没错。"秦寒道，"在下界的边缘，出现了空间裂缝。"

"空间裂缝？"玄天真人大惊。

"是的。"秦寒道，"而且，裂缝中渗出的气息，比魔界还要恐怖。"

"那是什么？"玄天真人问。

"不确定。"秦寒摇头，"但我怀疑，是另一个异界。"

"异界……"玄天真人脸色惨白。

魔界已经够可怕了。

如果再来一个异界，下界必亡。

"我们要做好准备。"秦寒道。

"我明白。"玄天真人点头。

"通知所有宗门，集结兵力。"

"是。"

接下来的日子，整个下界再次备战。

但这次，大家都很有信心。

因为有秦寒。

化神期的秦寒，是下界的最强战力。

"有他在，下界不会亡。"众修士道。

秦寒却不敢大意。

他深知，异界的恐怖，远超想象。

"必须更小心……"秦寒喃喃。

# 第二百零四章 异界降临

时间：备战期间
地点：下界·边缘地带
人物：秦寒、联盟大军

就在下界备战之时。

一个消息传来。

空间裂缝扩大了！

而且，从裂缝中，涌出了无数怪物。

这些怪物与魔界的魔物截然不同。

它们的身体半透明，仿佛由虚空构成。

"这是……虚空生物？"秦寒凝重。

他派出的神识分身，正在侦查裂缝。

"必须立即行动。"秦寒道。

他召集联盟大军。

五万修士，赶往裂缝所在地。

秦寒一马当先，冲在最前方。

"挡住裂缝！"

联盟大军布阵，将裂缝包围。

但虚空生物的数量太多了。

如潮水般涌出，无穷无尽。

"太多了……"林玄绝望。

"别慌。"秦寒道。

他全力出手，化神期的神通全力爆发。

"空间封锁！"

秦寒利用空间法则，封锁了裂缝。

裂缝暂时停止涌出怪物。

"快！消灭已经涌出的怪物！"

联盟大军齐声怒吼，开始斩杀虚空生物。

这些怪物虽然奇怪，但实力不算强。

最强的也只有元婴期。

对现在的联盟来说，完全可以对付。

经过一天的战斗。

所有涌出的虚空生物，全部被消灭。

"成功了！"众人欢呼。

但秦寒没有放松。

他盯着被封印的空间裂缝，眼中闪过一丝担忧。

"这只是开始……"

果然，片刻后。

封印开始颤动。

一股恐怖的气息，从裂缝中传来。

"来了……"秦寒咬牙。

裂缝中，走出一个巨大的身影。

身高五十米，半透明的身体，散发着恐怖的虚空气息。

"虚空主宰……"秦寒凝重。

"而且，是化神后期！"

"该死……"秦寒脸色难看。

化神后期，比他强太多了。

"诸位，"秦寒道，"准备苦战。"

"是！"联盟修士齐声应道。

一场新的战斗，即将开始。

# 第二百零五章 苦战虚空主宰

时间：虚空主宰降临后
地点：下界·边缘地带
人物：秦寒、虚空主宰、联盟大军

虚空主宰居高临下，看着下界的修士。

"渺小的蝼蚁……"它开口，声音如雷。

"你们以为，能挡住本座？"

"能不能，试了才知道。"秦寒冷静道。

"狂妄！"虚空主宰怒了。

它一掌拍下，恐怖的虚空之力如海啸般涌来。

"挡住！"

秦寒全力出手，化神期的神通全力爆发。

"空间壁垒！"

一道空间屏障出现，抵挡虚空之力。

"轰！"

屏障碎裂，秦寒被震飞。

"呃！"

他喷出一口鲜血，气息衰败。

"化神初期……确实不够。"秦寒咬牙。

他准备燃烧寿元，换取更强的力量。

但就在这时——

联盟的修士们冲了上来。

"我们帮你！"林玄喝道。

"我们一起！"

数百名修士齐声怒吼，冲向虚空主宰。

"找死！"虚空主宰一掌拍出。

数百名修士当场毙命！

"不……"秦寒悲愤。

"别冲动。"玄天太上出现在他身边。

"太上长老？"秦寒惊讶。

"我来帮你。"玄天太上道。

"多谢前辈。"

两人联手，对付虚空主宰。

玄天太上是化神后期，秦寒是化神初期中期。

两人联手，勉强能与虚空主宰对抗。

战斗持续了三天。

双方打得难解难分。

虚空主宰虽然强，但也无法快速击败两人。

"该死的人类……"虚空主宰愤怒。

它燃烧虚空之力，实力暴涨。

化神后期圆满！

"完了……"秦寒脸色惨白。

"别怕。"玄天太上道。

"我也燃烧寿元！"

两位强者同时燃烧寿元，实力暴涨。

秦寒达到化神初期后期，玄天太上达到化神后期圆满。

"现在，我们拼了！"

两人全力出手，轰向虚空主宰。

虚空主宰在两位强者的围攻下，渐渐落了下风。

"该死……"虚空主宰咬牙。

"本座还会回来的！"

它留下狠话，撤回空间裂缝。

"追！"秦寒道。

"不。"玄天太上拦住他，"那裂缝通往虚空界，我们追进去就是送死。"

"那怎么办？"

"封印裂缝。"玄天太上道。

"好。"

两人联手，封印了空间裂缝。

"成功了。"秦寒松了口气。

但代价很大。

联盟又损失了数千名修士。

而且，玄天太上燃烧了百年寿元。

"值得吗？"秦寒问。

"值得。"玄天太上道，"为了下界，百年寿元算什么。"

秦寒敬佩地看着这位老者。

"我会记住您的恩情。"

"应该的。"玄天太上微笑。

两人返回太华宗。

这一战，秦寒虽然没赢，但也学到了很多。

"化神后期……我还需要更强。"

他暗下决心，要继续提升实力。
