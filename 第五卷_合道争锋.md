# 第201章:合道期的感悟

**时间**:救出母亲后
**地点**:小世界
**境界**:合道期初期
**字数**:3020字

秦寒在小世界的山巅闭目而坐，感悟着合道期的力量。

合道期，与之前的境界完全不同。

返虚期是破碎虚空，在虚空中创造。

而合道期，则是身融天道，与天地万物合而为一。

"这就是合道期吗？"秦寒喃喃自语。

他能感受到，自己与整个天地融为了一体。

一举一动，都能引动天地之力。

一言一语，都能改变天地规则。

"言出法随……"秦寒睁开眼睛，眼中闪过一丝明悟。

他抬手一挥："起！"

面前的土地应声而起，化作一座山峰。

"落！"

山峰又缓缓落下，恢复原状。

"这就是合道期的力量。"秦寒喃喃自语。

命轮眼也随着修为的提升而进化。

现在已经是第七重境界，可以看到更多的未来可能性。

"母亲救出来了，父亲也团聚了。"秦寒感慨道，"接下来，我该做什么呢？"

他想了很久，终于找到了答案。

"我要让所有爱我的人，都能幸福。"

"我要让整个世界，都变得更加美好。"

"我要超脱天道，掌握真正的大道。"

秦寒站起身，眼中闪过坚定的光芒。

"合道期，还只是开始。"

# 第202章:天界的局势

**时间**:感悟后
**地点**:仙宫
**境界**:合道期初期
**字数**:3080字

秦寒返回仙宫，了解天界的最新局势。

自从他击败天帝救出母亲后，天界的格局发生了巨大变化。

天庭的威信大减，很多势力开始脱离天庭的控制。

仙宫的地位则大幅提升，成为天界最有影响力的势力之一。

"秦宫主，您回来了。"仙宫的长老恭敬地说。

秦寒在击败天帝后，被推举为仙宫的副宫主。

"天界最近怎么样？"秦寒问道。

"很混乱。"长老说，"天庭虽然实力受损，但还是天界最强的势力。很多小势力都在观望，不知道该投靠哪一边。"

"嗯。"秦寒点了点头，"天帝肯定不甘心失败，一定会想办法报复。"

"正是如此。"长老说，"所以我们更要小心。"

秦寒沉思片刻："我需要提升仙宫的实力，让仙宫成为天界真正的第一势力。"

"这……这很难。"长老说，"天庭底蕴深厚，不是一朝一夕能超越的。"

"没有什么是不可能的。"秦寒坚定地说。

接下来的日子里，秦寒开始在仙宫推行改革。

他开放了仙宫的藏经阁，让所有仙宫弟子都能学习高级功法。

他建立了奖励机制，鼓励仙宫弟子努力修炼。

他还将从下界带来的功法整理出来，与仙宫的功法融合，创造出更强大的修炼体系。

"仙宫的实力提升很快。"长老感叹道。

十年后，仙宫的实力翻了一倍。

已经有数十名返虚期强者，甚至还有几名合道期初期。

"现在的仙宫，已经可以与天庭抗衡了。"秦寒满意地点了点头。

# 第203章:小世界的完善

**时间**:仙宫事务稳定后
**地点**:小世界
**境界**:合道期初期巅峰
**字数**:3120字

秦寒回到小世界，继续完善这个世界。

现在的小世界已经有十万里的直径，山川河流，草木动物，一应俱全。

"但还是不够完美。"秦寒喃喃自语。

他抬手一挥，开始创造新的东西。

"要有日月！"

两团光芒出现在天空中，一为太阳，一为月亮。

"要有四季！"

小世界开始出现季节变化，春夏秋冬循环不息。

"要有灵智！"

小世界中的动物开始进化，产生了灵智。

"这就是一个完整的世界了。"秦寒满意地点了点头。

他来到父母居住的地方。

秦大江和秦仙姬正在一处精美的庭院中喝茶。

"寒儿。"秦仙姬看到秦寒，露出慈祥的笑容。

"母亲，父亲。"秦寒行礼。

"不用多礼。"秦大江笑着说，"这里没有外人。"

"父亲，母亲，我在小世界中创造了一个完整的世界。"秦寒说。

"我们看到了。"秦仙姬说，"这里很美，比天界还要好。"

"母亲喜欢就好。"秦寒说。

"寒儿，你辛苦了。"秦仙姬说，"为了救我和你父亲，你一定付出了很多。"

"这是我应该做的。"秦寒说。

"不。"秦仙姬摇了摇头，"你已经做得够多了。接下来，你要为自己而活。"

"母亲……"秦寒有些感动。

"听母亲的话。"秦仙姬说，"不要太累了。"

秦寒点了点头："我会的。"

接下来的日子里，秦寒一边陪伴父母，一边继续完善小世界。

小世界越来越完美，渐渐可以自我运转。

"这样的生活，真好。"秦寒感叹道。

# 第204章:苏清歌的飞升

**地点**:飞升台
**境界**:合道期初期巅峰

"清歌要来了。"秦寒露出开心的笑容。

他来到天界的飞升台，等待苏清歌的飞升。

飞升台是天界专门为下界飞升者准备的地方。

不一会儿，飞升通道中传来强大的气息。

一道身影从飞升通道中飞出，正是苏清歌。

"秦寒！"苏清歌看到秦寒，眼中闪过激动的光芒。

"清歌！"秦寒走上前，紧紧抱住她。

"我好想你。"苏清歌在秦寒怀里说道。

"我也是。"秦寒说。

两人相拥良久，才分开。

"你现在的修为是返虚中期，进步很快。"秦寒说。

"都是为了能早日飞升上来见你。"苏清歌笑着说。

"辛苦你了。"秦寒轻抚着她的长发。

"不辛苦。"苏清歌说，"只要能和你在一起，什么苦都值得。"

秦寒握住她的手："走，我带你去看看我在天界的住处。"

两人一起来到仙宫，秦寒为苏清歌安排了一处精美的庭院。

"这里很美。"苏清歌说。

"喜欢就好。"秦寒说。

接下来，秦寒带苏清歌参观了小世界。

"这是……"苏清歌震惊地看着小世界，"你开辟的世界？"

"是的。"秦寒点了点头，"以后你也可以住在这里。"

"太神奇了。"苏清歌感叹道。

两人又参观了小世界的各个角落，最后来到秦大江和秦仙姬的住处。

"父亲，母亲，这是苏清歌。"秦寒介绍道。

"清歌见过伯父、伯母。"苏清歌恭敬地行礼。

"好孩子。"秦仙姬笑着说，"寒儿能遇到你，是他的福气。"

"伯母过奖了。"苏清歌说。

一家人相谈甚欢，气氛融洽。

"这样的日子，真好。"秦寒感叹道。
